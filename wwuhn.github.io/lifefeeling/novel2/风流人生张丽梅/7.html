﻿（二十一）

就在１９９３年１月１９日，阴历腊月二十七的早上，大街上许多地方张灯结彩，各家各户都忙着烹炸食品，许多人脸上挂着欢乐的色彩，四处采购准备过春节的各种东西时，张丽梅的父亲却往科里给我打了个电话，说她得了急性肠胃炎，昨天晚上已送县医院住院治疗，让我有空了就赶快过去一趟。

我听自己的小心肝突然有恙，急得立刻向科里请了两天假，心如火焚地骑了车子，买了些她可能需要的物品以后，一会儿工夫就赶到了医院内科住院部。

当我小心翼翼地推开病房门，张丽梅看到我关切慈爱的目光，一大堆各式各样的食品和物品时，眼泪一下子就涌出了她眼眶，紧接着她就用双手捂住了十分憔悴的脸庞，轻轻抖动着纤细的身子，悄无声息地哭了起来。

当我硬拉着张丽梅的父亲到住院部走廊尽头问其原因时，他嘴里面吱唔了半天，我也没听出什么所以然来，只好像是吃了什么东西才出现了这种情况。等我再声色俱厉地责问他：“你有梅梅这么又听话又懂事的好女儿，本身就是个福，你和你老婆就应该好好珍惜才是。反过来大过年的，却怎么让她在这个时候住院了呢？”

张丽梅的父亲看我如此气势凶凶，眼睛先胆怯地盯着我看了一会后，就嘟嘟囔囔地给我说他俩口子都是从农村招的工，因为没什么高的文化和大的本事，所以工资就特别低。再加上他和老婆的父母受旧风俗的影响，都重男轻女，所以他俩口子也对张丽梅从小不感兴趣，就这住院的一千元押金，也都是他俩口子东借西挪着才凑出来的。

我看到张丽梅的父亲那副猥琐不展挂的龟样子以后，不由地就对他瞪着眼睛说：“梅梅长得那么聪明可爱，现在却让她住到了医院里边，身体也成了这副样子不说，吃的和换洗的东西都没有，任何人看了都觉得心疼和可怜，你们做父母的难道心里头就不感到难受和不安吗？”

张丽梅的父亲听我把这话说完之后，就用白眼仁把我不服气地翻了一下，然后用了朝鲜故事影片《鲜花盛开的村庄》里的一句话，对我表情很冷漠，对张丽梅却很无情地说道：“梅梅模样虽然长得好看，难道漂亮的脸蛋上能种出大米来吗？现在干啥都得要钱，她住院借的那些钱，我都不知道啥时候能还上呢？何况我那个宝贝儿子将来上大学，盖房子，做家具，娶媳妇，钱又从哪里来？

难道让我去抢银行不成？梅梅那个吃闲饭的累赘货你既然认为她各方面都特别好，那你就干脆认她当女儿算了。出院以后就直接住到你那阔家里，我们俩口子今后也就不cao这个闲龟心了好不好？“

看到张丽梅的父亲如此绝情寡义，我禁不住就对他气恨恨地说道：“你们这俩口子的确是我见过的人里边最最有出息的人了。既然你今天能说这样的话，那好，等一会我俩给梅梅打个招呼后，我就到你家里去上一趟，当着你那个老婆的面写个字据，你俩口子和我都签名以后，梅梅就全归我管算了。至于你这俩口子嘛！仔细照看着你的宝贝儿子不要出什么意外，以后也就当个现世宝一样的好好供着，从此就再不用cao梅梅这个累赘货的闲心如何？”

张丽梅的父亲听我这么一说后，立刻就如释重负般长出了一口气，连连点着头恬笑着对我说：“行！行！行！只要我们签过字后，咱们男子汉吐口唾沫都能把地上砸个坑，梅梅以后我可就再也不管了。她如果住院继续花钱，那可都是你一个人掏，我反正是一个子儿都不掏的。”

我看着张丽梅父亲的那个龟怂样子，我只说了句“你这人如果成了男子汉的话，狗肉都可以上席了”以后，就将他拉到了病房里边。

等张丽梅的父亲结结巴巴地把事情给张丽梅当着面说完，转身往外走时，我就看到张丽梅用牙紧咬着下嘴唇，美丽的杏核眼里立刻就闪射出了一道特别憎恶的寒光。

等我到了张丽梅的家里，让她父母亲把字据一式两份写好，大家都签名盖章完，我再把张丽梅住院的各样手续要到手后临出门时，就狠狠地往地上啐了一口唾沫，对送我到门口的张丽梅的父母非常藐视地说：“像你们这样当父母的货，真算是小**巴摇铃子——顶了个人名字。以后假如想有什么好出息的话，还不如到农村去找个烂牛逼，把头往里面塞上几十下才有那个可能，否则你们白日里光着屁股做千秋大梦还差不了多少。”

当我赶到医院，通过在外科住院部当护士长的当年插友安然，又给个别管事的人说了些好话，稍微塞了些钱后，就给张丽梅在内科住院部走廊的最里面，单独弄了个安静干净的小病房。

接着我又把张丽梅脏污成一团的衣裤洗净晾晒好，回来再把病房的门紧紧关上，给她用热水擦了脸和整个身子，又倒了些干净热水，为她把下身和拉稀屎显得特别红肿的屁眼完全擦洗干净后，时间都快到十一点了。

等我给张丽梅光着的屁股下面垫了些卫生纸，盖好被子，坐到病床上，用手抚摩着她憔悴的鸭蛋脸，抽着烟休息的时候，她看到我满脸流着的汗珠，立刻流着眼泪拉着我的手，哽咽着嗓音对我说：“爸，我和你肏了这么几年，到今天总算得到了好的回报。以后我不但可以永久地住在你家里，而且还可以和你生活在一起，对外也可以名正言顺地叫你爸。

我一想到自己成了你的女儿，心里面就感到特别高兴不说，还觉得有一些难受。

高兴的是我从此有了个好爸爸，惭愧的是这几年，自己光顾了在你面前撒娇着玩，再没有给你找个新的姑娘肏上一下。爸，等我病好了以后只要有机会，一定给你多找几个姑娘肏，只有用这个来表达我对你一份孝心，才能说明我这女儿真正爱你。

爸，你如果现在就想肏我的话，虽然我的身子还很虚弱，自己也不能动弹着帮你什么忙。不过你可以像你喜欢肏我的姿势一样，在床边上肏上我一阵也行。

假如你怕有人看见了不太那么方便，那你就用手摸我的逼，用指头戳逼里面也可以。反正晚上你肯定在这里陪床，到那时候再随便肏我也完全可以。只要你老人家心里高兴了，我这个当女儿的自然也高兴。“

自从进入腊月十五以后，张丽梅的父亲就叫她回家帮着准备年货，由于好长时间没有和她亲近了，我确实也有那个强烈的要求。但大白天的在这儿肏她我实在是没有那个胆量，所以我就往她的身边又移了一下，小心地将右手伸到被子里后，就摸起了她现在已长成了包子一样大的两个乳房，细滑绵软的小腹，刚长出的两片绵软柔嫩的小阴唇。

当我摸了一会儿，问张丽梅怎么得上急性肠胃炎的原因时，她立刻就沉下了脸，特别生气地对我说道：“爸，自从我到自己家里以后，好多天都没有吃上口肉，吃的全是他们吃剩下的那些东西。昨晚上吃饭的时候，我妈就端出了一盘剩了三天，我当时要吃不给我吃的回锅肉叫我吃。我端起盘子闻了一下后，就感到里面有股刺鼻的溲味，于是我就把它放到了桌子上不敢吃了。

谁知我爸看到这一情况后，立刻就瞪着大眼对我说：“这么好的肉菜，平时你就想吃的要命，现在你妈把它拿出来让你吃，你又挑三拣四的不想吃了。你难道是有钱人家的千金？还是哪个大领导的宝贝？你自己也不撒泡尿照照看你那个怂命样子，里面有个你吃好东西的影吗？

大冬天的菜怎么溲了？溲了又怎么啦？倒进锅里面随便热一下后，细菌早杀得没有影了。你害怕吃了肚子疼，娃娃家消化能力强得就像驴一样，随便跳上几个蹦子就没有了。说不定晚上临睡觉时你还嫌那盘回锅肉有些少呢！‘

就这样我忍着溲味把回锅肉吃了下去，果然没有过上多长时间，我就肚子疼得直不起腰来。我爸看到后又说我肚子假如疼的话，到卫生间随便拉上一泡屎就完全好了。等我忍着疼来到卫生间坐到马桶上的时候，那稀屎就像拉不完似的一个劲地往外直流。我只要感觉好像没有了，刚一直腰，稀屎就随后从屁眼里面扑哧扑哧地冒了出来。

我拉得浑身没劲两腿直抖时，胃里面又翻江倒海似的恶心了起来，等我忍了好几次没有忍住后，嘴一张就把卫生间吐了个一塌糊涂。

我爸和我妈听到动静赶到卫生间，看到我那个难受样子以后，这才把我用自行车驮到了医院。就这样子我妈还说我把卫生间弄了那么脏，到时候还要害她冲洗多少遍，我爸却尽说没有钱什么的牢骚话让我听个没完。

爸，你说我这样狠心的亲生父母，我跟他们在一起生活，还有什么生活兴趣和生命保证？所以我从今以后就决定再也不进那个家门，一心一意地和你和新妈一起生活，将来长大再好好地孝顺你们一辈子。“

我看张丽梅本来就蜡黄的脸上，由于说她父母的时候都已经有些变青，指头捏着她薄薄的小阴唇随便在玩，嘴里自然非常关切地对她说：“梅梅，你这个小心肝都已经把我伺候得够好的了。现在你既然是病人，那就先安心把自己的病完全养好，想吃什么东西了就吭个声，心里再不要胡思乱想。反正我们都已经成了一家人了，至于其它的那些事情，以后再说也不算怎么迟。”

张丽梅看我到这时了还这么关心她，漂亮的鸭蛋脸上一下子泛起了一片淡色红晕，神情非常羞赧地对我说：“爸，昨晚上我拉稀屎的时候，到最后连屁眼都拉得翻出来了，逼上都不小心糊了好多的屎。

因为我妈到现在都没有到医院来看过我，所以到早上的时候，屎都在逼口和屁眼周围干成了片片，而且还难受地特别要命。我一个姑娘家又不想让那个没用的爸洗上一下，自己想用水洗的话，病房里又那么多的病人，我自己身上又软得一点劲都没有，所以当时都快把我愁死了。幸亏你今天赶来了不说，还给我换了个这么好的单人房间，不嫌脏臭地给我把逼口和屁眼洗了个干干净净，要不然我都不知道怎么办才好。

爸，我自从看到你以后，心里面现在感觉舒畅多了。尤其知道自己已经成了你真正的女儿，再加上你把我的逼洗干净后又摸了一阵，我也不知道怎么搞的，逼里面不由地就痒了起来。爸呀！既然我的逼让你彻底舒服，那你就用两根指头给我戳一下了，你自己感到美就行。“

虽然自己在这个地方随意玩弄张丽梅的下身也不太方便，但我看到她那美丽的杏核眼露出的渴望眼神，期待用自己稚嫩身子报答我的一片真情，又想到她这样病了以后，也极需要在身心上得到某种慰籍和鼓励，我只好又往她身边挪了一下后，就用两根指头在她逼里面缓慢而小心地抽插了起来。

到底是彼此之间心心相印，就在我把张丽梅刚抽插了一阵，她满脸浮现出了灿烂的桃花，不断地挺动着屁股，小嘴微微张开，红豆似的柔嫩红润小鼻孔翕动着喘粗气时，病房的门突然被端药进来的小护士给一下子推开了。

我情急之下来不及拿出作孽的手，只好赶快移到了张丽梅的小腹上，就在那里装模作样地揉搓了起来。

那个小护士一眼看到这一情景后，粉嫩的娃娃脸立刻红了一片，紧接着就用飞快熟练的动作，给张丽梅在屁股上打了一针，然后眼睛把我意味深长地看了一眼后，没有吭声，端着药盘子走出了病房。

大约快到中午十二点的时候，张丽梅的班主任李娜和几个女同学来看她。当寂静的病房里面响起了她们叽里呱啦的说话声时，那个给张丽梅打针的小护士进来把我喊了出去，就说有些事情需要和我到医务室详谈一下。

当我和那个小护士一同来到医务室把门关好以后，她那炯炯有神的大眼睛就盯着我说：“华师傅，我听说那个小姑娘确实命苦，自打昨晚送来住院以后，脸色黄得都有些吓人不说，一个晚上都没有什么人来陪床，排泄物把衣裤糊了那么脏，也没有一个人给她洗上一下，整个情景可怜得就没法形容。

今天你这个好心人来了之后，不但将她认作了自己女儿，而且还给她换了个寂静的小病房，弄脏的那些衣裤也洗了个干干净净。华师傅，你这样的好人在现今的社会里面，可确实很难见到的活雷锋啊！“

我听了小护士说的话，又看到她模样长得也还甜纯可爱，就故作清高地对她说：“我哪是什么活雷锋呀！自己只不过是为人民服务了一下嘛！尽管社会是这个样子，现在的人也喜欢顾自己的那个小圈圈，但里面也有个别好心人嘛！我这样做既是本分也是应该，并没有和其它好人有任何不同的地方。”

小护士听我把这些屁话刚刚说完，粉嫩的娃娃脸立刻红到了耳根后，就用眼睛把我瞅了一下，跟着就小声说：“华师傅，既然你有这么个难得的好心，那你刚才在那个小姑娘的下身摸什么呢？”

我看这个小护士眼睛里流露着急切想知道答案的神色，正要张嘴给她说个所以然时，就有个小护士把门一下推开，紧接着对她大声说：“杨漾，现在已经下班了，我在外面等你，你收拾一下东西，我们就一起回家。”

杨漾看到这种情况，只好把我又瞅了一眼后，就小声对我说：“华师傅，我今晚上十二点开始值夜班，你如果给那个小姑娘陪床的话，我们到那时候再细聊好不好呀？”

我随着杨漾的话音点了一下头后，就走出医务室到了住院部办公室。

当我在住院部办公室给家里打了个电话，告诉老婆今早上发生的所有事情以后，一点刚过她就提着饭盒来到了病房。

等到我和张丽梅吃完饭，坐在病床旁边的方凳上抽烟时，老婆一面收拾着碗筷，一面脸上洋溢着信服的神色说：“老头子，其实我早就想要梅梅当我们的亲女儿了，只是因为条件不允许罢了。这次老天爷给了我们这么好的一个机会，你又当机立断地解决了这个问题，真不愧为是我的好老头子。

这次梅梅得病身体亏得很厉害，我工作忙的也顾不上照应她，你就多辛苦一点，白天晚上都在这儿照料她，争取在春节前梅梅能够出院，我们一家人也好过一个欢欢乐乐的春节。如果有什么事情需要办的话，你就给我打电话好了。“

老婆说完这些话以后，就用手抚摩着张丽梅的脸，非常慈爱地说了几句安心养病，想吃想要什么就给新爸说的话后，就提着饭盒出了病房门。

当不会骑车的老婆把我的自行车推走了以后，趁着住院部走廊里又特别安静无人的时机，张丽梅就欠起身子，让我把她抱到怀里，她则将头靠在了我的胸脯上，一面把手伸到我的裤子里，温柔地抚摩着我的龟和卵蛋，一面仰着脸对我说道：“爸，我今天因祸得福，心里面觉得都幸福死了。李老师和几个女同学来看我，听我说了事情的整个经过以后，她们都把我羡慕得了不得。

尤其是同学赵小玲，在李老师和其它几个女同学走了以后，她又陪着我待了好一会，给我说她如果有你这么好的一个爸爸，一天到晚的心疼她该有多好后，这才告别我回家去了。

爸，我看小玲长得也很美，脸形和我一样不说，眼睛大大的也很有光彩。假如你愿意的话，哪一天我把她叫到家里后，如果各个方面的条件许可，你就把她给肏了算了。我估计她只要尝到肏逼的那个甜头，你再给她些好处，她也不会随便向外面说些什么。“
