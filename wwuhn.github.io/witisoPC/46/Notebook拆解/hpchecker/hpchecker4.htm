    正如上页说的，笔记本一般是勺状的结构，D面相当于最底下，想知道内容还需要从上面来，于是我们本页继续拆C面，这回多了很多排线、接口，所以用到镊子的部分会比较多，并且显示器也是这个时候来拆。


键盘要先取下来，如图用镊子轻轻撬开


键盘四个角都有这样的小突起


键盘下部有很多连接到显示器上的线


键盘的排线这样拆下来


平板状的排线连接


键盘已经被拿下来了（键盘上方的黑色触控按键所在地为多媒体面板）


沿着线卡取线，以免损伤线材


插头式连接要双向用力


屏幕线中最粗的一条，轻用力拔出


由于没有螺丝固定，C面的多媒体面板很容易就取下来了，下面是音箱和屏幕转轴架


多媒体面板下方两侧均设有2~3个不等的屏架加固螺丝，取下，分类放好


屏幕取下后C面显得极为整洁

    以上步骤结束之后，我们需要打开C面，因为最需要清理的主板和散热器还在里面，不过这步有一些注意的地方，一方面是螺丝的长短及位置要记清，另一部分就是C面取下时用力有一定的规律，我们接着看图说话。


螺丝依然要分类放好


C面边缘需要用镊子轻轻翘起，配合下面一步


手在中央处向上用力，配合前一步的边缘处理，轻松取下C面
