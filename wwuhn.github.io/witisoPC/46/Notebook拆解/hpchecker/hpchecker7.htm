    插入一嘴，前一页中我们提到了改善GPU得散热状况，是从外部条件来改善过热的情况，还忽略了一点，GPU本身也是可以升级的，惠普DV2000这款笔记本搭载的是NVIDIA的GF 7200独显，我们可以更换为内置温控功能并且性能领先于它的GF 7400显示核心，这样也可以解决GPU过热所带来的各类麻烦，不过这种芯片级的维修只有拥有BGA平台的维修站才可以实现，例如本次与杨仔“联合军演”的芯维芯片级笔记本维修车间。
