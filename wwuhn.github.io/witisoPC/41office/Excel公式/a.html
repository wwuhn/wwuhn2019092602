二、运算符优先级

公式中同时用到了多个运算符，Excel将按下面的顺序进行运算：

  1. 如果公式中包含了相同优先级的运算符，Excel将从左到右进行计算；
  2. 如果公式中有不同级别的混合运算，运算符的顺序从高到低依次为：引用运算符（:）、负号（如一l）、%（百分比）、^（乘方）、*和/（乘和除）、十和一（加和减）、&（连接符）、比较运算符。
  3. 如果要修改计算的顺序，应把公式需要首先计算的部分括在圆括号内。

三、相对引用于绝对引用

1、概念：

  1. A1 : 相对引用。
  2. $A$1: 绝对引用
  3. $A1: 列绝对引用，行相对引用
  4. A$1: 行绝对引用，列相对引用

2、用途:

  1. 行前添加$，复制公式时，行数不发生变化。
  2. 列前添加$，复制公式时，列数不发生变化。

3、F4键——引用方式之间切换的快捷键：

  1. 按一次：绝对引用；
  2. 按两次：对行绝对引用、对列相对引用；
  3. 按三次：对行相对引用、对列绝对引用；
  4. 按四次：相对引用。

4、举例：

  1. 九九乘法表的生成，灵活运用了引用方式。

  1. VLOOKUP多行多列查找也运用了多种引用方式。
