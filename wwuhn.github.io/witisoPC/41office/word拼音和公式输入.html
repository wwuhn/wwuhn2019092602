1 可以插入现成的公式，然后可考虑修改；
2 插入-公式-插入新公式
3 Word中的编辑好的公式可以做为图片粘贴到excel
利用数学输入面板进行输入公式
打开方式：搜索---数学---数学输入面板；
附件-数学输入面板
什么是数学输入面板？数学输入面板使用内置于 Windows 7 的数学识别器来识别手写的数学表达式。然后可以将识别的数学表达式插入字处理程序或计程序。
点击“插入”可以将识别的数学表达式插入字处理程序或计算程序。数学输入面板只能将数学表达式插入支持数学标记语言 (MathML) 的程序中。
打开word,再打开数学输入面板，输入公式后，点击插入，即可将公式插入Word.
