使用动态 UCS 功能

可以在创建对象时使 UCS 的 XY 平面自动与实体模型上的平面临时对齐。使用绘图命令时，可以通过在面的一条边上移动指针对齐 UCS，而无需使用 UCS命令。结束该命令后，UCS 将恢复到其上一个位置和方向。

动态UCS 的X 轴沿面的一条边定位，且X 轴的正向始终指向屏幕的右半部分。动态 UCS 仅能检测到实体的前向面。

可以使用动态 UCS 的命令类型包括：■ 简单几何图形。直线、多段线、矩形、圆弧、圆■ 文字。文字、多行文字、表格■ 参照。插入、外部参照■ 实体。原型和 POLYSOLID■ 编辑。旋转、镜像、对齐■ 其他。UCS、区域、夹点工具操作提示通过打开动态 UCS 功能，然后使用 UCS 命令定位实体模型上某个平面的原点，可以轻松地将 UCS 与该平面对齐。通过按 F6 键或 SHIFT+Z 组合键可以临时关闭动态 UCS。

坐标标注

坐标标注(Dimordinate）命令用于自动测量并沿一条简单的引线显示指定点的X 或Y 坐标(采用绝对坐标值)。

图形的精确定位：坐标（相对坐标和极坐标）：

a 相对坐标和极坐标：如相对于长方形（100*100）上的左下角的@155,155的位置，绘制一条呈22度度的长度为1000的直线：L；shift+右键，F，点击左下角，输入：@155,155,再输入：1000<45;

b 极坐标：是通过一个长度值和一个角度值来定位的；如果选择了正交模式，则只需要输入长度值，对应有角度是90度角，如果选择的是极轴模式，则也只需要输入长度值，但对应的角是捕捉到的角度值（两端会有列限延长的虚线，可设置增量角）；

c 0度角的方向是东，即为X轴正向，角度的正增量方向为逆时针counter clockwise方向；

e 绘制图形（特别是多线段的圆弧）时，光标停的方向，位置，捕捉的点会影响到绘制图形的方向，形状等；多线段画弧时不要启用DYN功能；

捕捉、自动捕捉和临时捕捉

a 捕捉snap：用于设置鼠标光标的移动间距，有时因间距设置较大，又应用了捕捉功能，所以光标老是捕捉不到点；有没有设置此功能的特征是：光标是否按间距移动；捕捉模式用于限制十字光标，使其按照用户定义的间距移动。当“捕捉”模式打开时，光标似乎附着或捕捉到不可见的栅格。捕捉模式有助于使用箭头键或定点设备来精确地定位点。

b 自动捕捉(对象捕捉)：状态栏按钮控制和草图设置；

在绘图过程中，经常需要指定一些对象上已有的点，例如端点，圆心和两个对象的交汇处，如果只是凭观察来拾取，不准确，如果去计算坐标，则又太慢；在CAD中，通过设置对象捕捉功能和草图设置对话框等方式来调用对象捕捉；

c 临时捕捉：工具栏按钮或快捷键；相对于某a点开始某b点：命令（如圆）---捕捉自（对象捕捉工具栏）---点击A点---@：相对坐标（B点相对于A点的位置）；shift+单击；

目标捕捉(Osnap)是捕捉绘图区中对象实体的特征点，如端点、圆心点、交叉点等。而捕捉（SNAP）是捕捉绘图区的辅助栅格点。它们的作用是将十字光标强制性地准确定位在已存在的实体特定点或特定位置上。利用这一功能，可以绘制出很精确的工程图。需要强调的是，在发出对象捕捉命令前，首先必须要执行绘图和修改命令，然后才能使用对象捕捉和对象追踪命令来精确定位点。

正交、极轴、对象追踪

a 正交和极轴只能二选一，正交是固定90度角捕捉图像，极轴可以按设定的角度捕捉图像，会有一根无限拉长的虚线出现，当要确定一个点是，如果有一个出现捕捉点（不需点击），一条无限拉长的虚线形成的夹角，则只以输入一个长度值即可捕捉点和夹角定位一个点；

b 正交ortho，将定点设备的输入限制为水平或垂直；如画直线时只能90度绘制，确定第二点时，只需要输入距离值，按后按鼠标的方向确定是正90还是负90的极轴来确定点；

c 极轴

是指用状态栏的控制按钮和草图设置的值来控制给定角度的捕捉，并产生一条无限拉长的虚线；

d 对象追踪：

需先设置有对象捕捉功能，才对捕捉的对象产生一条无线拉长的虚线形成追踪；

极轴追踪可以按角度绘制对象（极坐标）（无限长的虚线），极轴追踪和对象追踪两者可同时使用，如果事先知道了要追踪的方向（角度），可以使用极轴模式，如果事先知道了与某种对象的关系，如相交，则可作用对象捕捉模式；

动态输入功能

动态输入功能，可以在指针位置处显示标注输入和命令提示信息；在输入多段线画圆弧时尽量不要使用此功能；

启用“动态输入”时，可以在光标附近的工具栏提示中输入坐标值。否则是在命令行中输入；

区别是用户的注意力可以保持在光标附近。动态输入不会取代命令窗口。您可以隐藏命令窗口以增加绘图屏幕区域，但是您在有些操作中还是需要显示命令窗口。“动态输入”有三个组件：指针输入、标注输入和动态提示标注输入

a 当启用指针输入且有命令在执行时，十字光标的位置将在光标附近的工具栏提示中显示为坐标。可以在工具栏提示中输入坐标值，而不用在命令行中输入。第二个点和后续点的默认设置为相对极坐标（对于RECTANG 命令，为相对笛卡尔坐标）。不需要输入 @ 符号。如果需要使用绝对坐标，请使用井号 (#) 前缀。例如，要将对象移到原点，请在提示输入第二个点时，输入#0,0。

b 启用标注输入时，当命令提示输入第二点时，工具栏提示将显示距离和角度值。在工具栏提示中的值将随着光标移动而改变。按 TAB 键可以移动到要更改的值。

标注输入可用于ARC、CIRCLE、ELLIPSE、LINE 和PLINE。使用夹点编辑对象时，标注输入工具栏提示可能会显示以下信息：旧的长度\移动夹点时更新的长度\长度的改变\角度\移动夹点时角度的变化\圆弧的半径

c 动态提示，提示会显示在光标附近的工具栏提示中。用户可以在工具栏提示（而不是在命令行）中输入响应。按下箭头键可以查看和选择选项。按上箭头键可以显示最近的输入。
