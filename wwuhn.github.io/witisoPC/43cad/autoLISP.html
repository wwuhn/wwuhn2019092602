List Processing Language

是一种程序设计语言，用于人工智能、机器人、专家系统、博弈、定理证明等领域；

被称为符号语言，因为它处理的对象是符号表达式，LISP的程序和数据都可以以符号表达式的形式来表示；

AutoLISP是为二次开发autocad而专门设计的编曲程语言，它起源于LISP语句，嵌入在autocad的内部。既有LISP人工智能的特性，又有CAD强大的图形编辑功能，可以氢AutoLISP程序和AutoCAD的绘图命令透明地结合起来；使设计和绘图完全融为一体。

利用autolisp可以进行各种工程的分析计算，自动绘制复制的图形，还可以定义新的autocad命令，驱动对话框、控制菜单、并且可为autocad扩充具有一定智能化参数的功能；

(defun c:pbox()

;以下 pa、ww、hh 值须由用户输入

(setq pa (getpoint "输入左下角点:"))

(setq ww (getdist pa "\n宽度ww:"))

(setq hh (getdist pa "\n高度hh:"))

;以下pb、pc、pd值须由程序依据pa、ww、hh求出

(setq pb (polar pa 0 ww))

(setq pc (polar pb (/ pi 2) hh))

(setq pd (polar pc pi ww))

;以下依据 pa、pb、pc、pd 点画矩形

(command  "pline" pa pb pc pd "c")

;以下mp1、mp2、mp3、mp4值须由程序根据pa、ww、hh

;& pb、pc、pd 值求出

(setq mp1 (polar pa 0 (/ ww 2)))

(setq mp2 (polar pb (/ pi 2) (/ hh 2)))

(setq mp3 (polar pd 0 (/ ww 2)))

(setq mp4 (polar pa (/ pi 2) (/ hh 2)))

;以下根据 mp1、mp2、mp3、mp4 点画出两条直线

(command  "line" mp1 mp3 "")

(command  "line" mp2 mp4 "")

)
