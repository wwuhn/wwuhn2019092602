Operators are special symbols that perform special operations on one,two,or three.

Assignment operators are evaluated form right to left.This operator can also be used on objects to assign object references.

The unary operators require only one operand,they perform various operations such as incrementing/decrementing a value by one,negating an expression on inverting the value of a boolean.+、++、－、－－、！。

The code result++,and ++result,will both end in result being incremented by one.The only difference is that ++result evaluates to the incemented value,whereas the result++evaluates to the original value.

the &&and ||operators perform conditiona-AND and conditiona-OR operators on two boolean oexpressions.

use the ?: operator instead of an if-then-else statement if it makes your code more readable.

The instanceof operator you can use it to test if an object is an instance of a class,an instance of a subclass,or an instance of a class that implement a particular interface.

Operators may be used in building expressions,which compute values,expressions are the core components of statements;statements may be grouped into blocks.

An expression consists of variables,operators and method invocations,which are constructed according to the syntax of the language,that evaluate to a single value.

The new operator returns a referenc to the object it created.

Java操作符是Java程序中对操作数（数据和对象）实施的操作，每个操作符都有确定的操作数的数目和功能；

表达式是变量、常量、运算符、方法等按照一定的运算规则组成的序列，并返回一个值，其返回值与操作数和操作符相关；

赋值运算符“＝”的作用是将数据写入到变量的贮存单元中，在“＝”的左边是变量，右边是待写入的数据值或引用值（可以是对象）

a+=b的理解：

1 用一个表达式的值赋值给一个变量；

2 变量是第一个字符a;

3 表达式是去掉"="剩下的部分；

按位操作符允许我们操作整数基本数据类型中的单个“比特”(bit),即二进制位。按位操作符会对两个参数中对应的位执行布尔代数运算；

按位操作符来源于C语言面向底层的操作，在这种操作中经常需要直接操作硬件，设置硬件寄存器内的二进制位。

Java的设计初衷是嵌入电视机机顶盒内，所以这种面向底层的操作仍被保留下来。

移位操作符操作的运算对象也是二进制的“位”。它们只可用来处理整数类型；

int y=1/3;         =y=0

操作符：操作一个或多个对象，半生成一个新值；

操作符可能会改变操作数自身的值，这被称为副作用；几乎所有的操作符只能操作“基本类型”，例外的是“="、"=="和"!=",它们能操作所有的对象，此外，String类支持"+"和"+=";

赋值"="取右边的值，把它赋给左边；右值可以是常数、变量或表达式，但左值必须是一个明确的，已命名的变量；

整数除法会去掉结果的小数位，而不是四舍五入的结果，Java也使用一种简化符号同时进行运算与赋值操作，这用操作符的后面跟一个等号表示；

逗号：for(int i=1,j=i+10;i<5;i++,j=i*2){}
