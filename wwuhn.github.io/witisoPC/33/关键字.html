修饰符 abstract 只能为父
修饰符 final 只能为子，表明它是无法扩展的，可者说我们无法用它来定义子类，将一个类声明为final可以避免对该进行不必要的扩展。Final tells that the variable will never change;
修饰符 native 表明声明的方法是一本地方法，这个本地方法是一个不是由Java语言编写的方法。可以通过JNI接口调用其他语言实现对操作系统底层的访问，如C语言的dll库等；
修饰符 static tells the compiler that there is exactly one copy of this variable is existence;用来定义类方法或类变量，或叫静态方法、静态属性；
修饰符 synchronized 声明方法的同步；
修饰符 volatile 是一个异步控制修饰符，在多线程中使用，使用该修饰符修饰的变量可保证各线程对该变量的访问的一致性；
异常处理 catch 可以是多个，一般异常处理范围较小的在前面。
异常处理 finally 一定执行的部分，如释放资源，关闭文件。
