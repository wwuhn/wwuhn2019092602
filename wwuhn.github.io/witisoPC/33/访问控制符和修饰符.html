访问控制符和修饰符modifier

当变量、方法和类被其中不同的关键字修饰时，会有不同的访问权限和使用范围。利用不同的访问控制符，可以使用方法将类中的数据成员隐藏起来，从而控制不同的用户对类的访问和修改。封装实际上就是通过方法将类的成员隐藏起来以控制用户对其访问和修改的程序。

用来修饰外部类的访问权限修饰符只能是友好(default)的和公共的(public),内部类可以是private,default,protected,public;

default可以被同一包中的其他类访问；

Controlling access to members of a class;

A class which has no modifier,its default is private.

private进行属性封装，setXxx()公有方法来设置类中的私有属性；

Java中提供了对类的成员的访问在四个不同的范围

1 同一类中；

2 同一包中；

3 不同的包中的子类；

4 不同的包中的非子类；

访问权限又可分为四种

1 public

2 protected

3 default

4 private

这四种访问权限在四个范围的情况是不同的；
