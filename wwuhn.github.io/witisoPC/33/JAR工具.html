JAR工具

发布类和应用程序（写清单文件略有区别）

1 编写一个源文件.java;

2 编译源文件；

3 写一个清单文件Manifest.mf；

4 在命令行创建jar文件；

Jar cfm test.jar manifest.mf *.class

(*.class部分也可以一个个class文件列出）

c:表示要生成一个新的JAR文件；

f:表示生成JAR文件的名字；

m:表示文件清单的名字；

发布类文件的manifext.ms

Manifest-Version: 1.0

Class:class1 class2(如果是发布应用程序，这里写成：Main-class: 有main方法的类名）

Created-By: wwu

（以上":"字符后都必须有一个空格）

发布类test.jar文件可以放到JDK安装目录下的jre\lib\ext目录中，这样就可以用该JAR文件中的类创建对象了；

发布应用程序test.jar可以在安装有JRE的计算机上用Java解释器执行或双击执行；

java -jar test.jar

1 查看JAR文件：jar tf test.jar

2 解压JAR文件：jar xf test.jar

3 更新JAR文件：jar ft test.jar
