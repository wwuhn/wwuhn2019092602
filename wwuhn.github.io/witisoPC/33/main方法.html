main方法

程序的进入点：main()方法；

public:main()方法要给外部的JVM程序调用；

static:一开始就要把执行的内容加载到内存中，并且持续到整个程序结束，当对象还没有产生时就已被JVM调用；

void:main()方法主要是执行程序，没必要有返回值；

String[]:执行时允许输入多个文字当作参数值（空格分隔）；

Main method's array is the mechanism through wihich the system passes information to your application.Each String in the array is called a command-line argument,command-line arguments let users affect the operation of the application without recompiling it.

Command-line arguments can accept any number of arguments from the commnad line.This allows the use to specify configuration information when the application is launched.The user enters command-line arguments and specifies them after the anme of the class to run.

main()方法的参数与一般方法不同，是该方法被系统自动调用时，无法在程序中给它传实参，但是，从CMD下运行程序时，可以将参数传递给它（参数用空格分隔）。

启动类startup class:含有main()方法的类，程序从main（）方法开始执行；
