Component 接下或释放鼠标键 KeyEvent

Jbutton、Button 单击按钮 ActionEvent 　
JCheckBox 单击复选框 ActionEVent、ItemEvent  
Jlist、List 选择选项 ListSelectionEVent  
JMenuItem 选择菜单项 ActionEvent、ItemEvent  
JRadioButton 单击选择按钮 ActionEvent、ItemEvent  
JTextField、 获得焦点后按回车键 ActionEvent 　
JSlider 滑块滑动 ChangeEvent 　
Window 窗口打开、关闭等 WindowEvent
