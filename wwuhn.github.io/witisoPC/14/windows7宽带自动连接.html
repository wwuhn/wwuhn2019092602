windows7 宽带自动连接

1 建立的宽带连接要勾选“记住密码”

查看建立的连接:控制面板\网络和 Internet\网络和共享中心---更改适配器设置:即可查看本地连接\宽带连接\无线连接

新建一个宽带连接：控制面板---网络和Internet---查看网络状态和任务---设置新的连接或网络---连接到Internet---您如何连接？：选择宽带---输入用户名、密码、宽带连接名称：BC，勾选：记住密码；

win+r---rasphone---新建:

2 Internet选项

internet选项---连接---选择BC---勾选：始终拨打默认连接；

3 宽带连接属性设置

控制面板---网络和共享中心---连接到网络---右下角弹出已有建立的连接的对话框---右击BC---属性---选项：去掉勾选：提示名称、密码和证书；

4 设置rasphone.exe自动运行

4.1 方式一：注册表设置

win+r---输入regedit---HKEY_LOCAL_MACHINE＼SOFTWARE＼Microsoft＼Windows＼CurrentVersion＼Run ---右边的窗口中新建一个字符串键，名字任取，双击此键输入键值为c:\Windows\System32\rasphone -d BC  (你的宽带连接的名称）；这样你可以在运行里输入msconfig找启动项看一下是不是加入了一个windows启动项，而它的命令里就是我们的c:＼Windows＼System32＼rasphone -d "BC"，这样就可以开机连网了

也可以是批处理的形式：新建一个批处理文件，输入以下内容：

reg add "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run" /v autodial /t REG_SZ /d "%SystemRoot%\system32 \rasphone.exe -d 宽带连接" /f

方式二：任务计划

必要条件：系统保存了宽带连接的用户名和密码

第一：在开始菜单的搜索框中输入”任务计划程序”(task schedule)命令，然后打开！

第二：在右侧的”操作”窗格中单击”创建基本任务”选项．打开输入名称和描述（比如：开机自动连接宽带）．

第三：单击下一步，在后面的步骤中将”触发器”选项设置为”当前用户登陆时”，下一步～～将”操作”选项设置为”启动程序”，将”程序或脚本”设置为” c:\Windows\System32\rasphone.exe -d bc”单击下一步．

第四：选中"当单击’完成’时，打开此任务的对话框"复选框，单击：“完成”．

第五：随后打开“开机自动宽带连接属性”对话框．切换至”触发器”(trigger)选项．单击下方的”编辑”．

第六：选择“延迟任务时间”复选框，手动将时间修改为5秒．不建议设置为1～2秒，3秒以上为好～因为猫跟电脑有连接的过程！

第七：单击确定．

第八：打开宽带连接属性对话框，切换到”选项，取消”提示名称密码和证书等（P)”复选框．(ncpa.cpl)

第九：打开IE,工具----Internet选项-----连接-----始终保持默认连接

以后登陆后，系统就会在5秒后自动连接了！

c:\\windows\\system32\\rasphone.exe是拨号程序的路径。

-d是指进行宽带连接，最后的“宽带连接”是要自动拨号的宽带连接名。

有很多软件可以开机自动拨号

target:C:\Windows\System32\rasphone.exe -d "broadband connection 3";

"broadband connection"右击---properties---options tab:1 display progress while connecting,when connectting ,display connection progress.2 prompt or name and password,certificate,etc.:display connection dialogue.

winr-rasphone--连接:即可打开连接窗口;

右下角的网络连接图标---右击-打开网络和共享中心-更改适配器设置-选择宽带连接图标右击-建立快捷方式;

一、首先在网络链接里创建一个pppoe的宽带连接并且手动拨号成功。

二、右键开始按钮---计算机管理---右键任务计划程序---创建基本任务---名称：拨号1，然后下一步---当前用户登录时，然后下一步---启动程序，然后下一步---在程序和脚本下面输入：rasdial 宽带连接你的账号密码，然后下一步---完成并确定。

三、重复第二步的步骤，只是把任务名称改为拨号2，然后修改任务属性，触发器---编辑---延迟任务时间8秒---确定。

四、这样就建立了两个自动拨号任务，第一个会651错误失败，但8秒后的第二个拨号任务会成功，开机或重启不需要任何额外操作就可以自动联网了。
