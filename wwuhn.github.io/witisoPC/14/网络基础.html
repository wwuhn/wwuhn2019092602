ipc$

是internet process connection的缩写，它是windows主机之间通信的管道。名称后面附加$,则表示该管道不呈现给普通用户，只隐藏在操作系统中。默认状态下，管道使用139、445端口侦听其他电脑发出的连接请求。

利用IPC$,系统管理员可以方便地管理没有开启任何共享文档的远程主机，但是IPC$同样会被入侵者利用，如远程添加帐户、修改帐户权限及非法访问用户的数据文件。

根据连接权限的不同，IPC$连接可以分为IPC$空连接和IPC$特定权限连接两种：

1 IPC$空连接：只需目标主机没有取消默认的IPC$管道，并且server服务处于运行状态即可。

2 IPC$特定权限连接：在IPC$空连接的基础上附加可用的帐户和密码。

1 IPC$空连接：只需目标主机没有取消默认的IPC$管道，并且server服务处于运行状态即可。

net use \\目标IP地址\ipc$ "" /user:""

IPC$空连接无法进行任何管理操作，如传输文件、执行程序，但是，它可以提供少量主机信息，如获得共享文件夹列表，具有编程基础的黑客还可以通过脚本获得目标主机的用户帐户列表，这样，将大大降低暴力密码破解的难度。

例1 与192.168.11.66建立IPC$空连接，获得共享文件夹列表

net use \\192.168.1.66\ipc$

net view \\192.168.1.66

2 IPC$特定权限连接：在IPC$空连接的基础上附加可用的帐户和密码。

net use \\目标IP地址\ipc$ pw /user:username

例2 与192.168.1.66建立IPC$特定权限连接，并复制文件至远端电脑中的c:\video目录。

net use \\192.168.1.66\ipc$ password /user:clamp

    Windows 7中的网络类型

    Windows 7中的网络类型分为三种，实质是两种：可信任网络和不可信任网络。其差异在于防火墙的策略和文件共享等功能的配置。

    windows7会自动记录当前的网络配置及相关信息，并在用户下次连接到此网络中时自动匹配及切换相应网络配置文件；

    家庭网络和工作网络同为可信任网络，选择这两种网络类型会自动应用比较松散的防火墙策略，从而实现在局域网中共享文件、打印机、流媒体等功能。但当用户选择家庭网络时Windows 7会自动进行“家庭组”的配置，比如检测局域网中是否存在家庭组、配置家庭组中的共享设置、加入家庭组等。

    家庭网络和工作网络中的计算机通过受保护设备（如防火墙和路由器）连接到internet;

    公用网络为不可信任网络，选择公用网络则会在Windows防火墙中自动应用较为严格的防火墙策略，从而到达在公共区域保护计算机不受外来计算机的侵入。

    如果您知道不需要共享文件或打印机，则最安全的选项是公用网络。
