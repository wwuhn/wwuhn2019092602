MySQL连接并插入数据

下述程序在Jcreator运行，需在Jcreator做如下设置：

1 "D:\java\jdk\lib\mysql-connector-java-5.1.22-bin.jar"

2 配置→选项→JDK配置文件，选中“JDK版本”，编辑→“类”选项卡：添加→添加存档→选择上述路径的驱动；

如果在文本模式下（如DOS环境）编译与执行，都必须使用"classpath"参数指定JDBC Driver的JAR文件位置。假设

1 要编译的Java文件路径是：c:\ConnectTest_MySQL.java

2 JDBC Driver文件路径D:\java\jdk\lib\mysql-connector-java-5.1.22-bin.jar；

编译

c:\>javac -classpath .;D:\java\jdk\lib\mysql-connector-java-5.1.22-bin.jar ConnectTest_MySQL.java

执行

c:\>java -classpath .;D:\java\jdk\lib\mysql-connector-java-5.1.22-bin.jar ConnectTest_MySQL

以下是java应用程序连接的代码（区别网页连接）

import java.sql.*;

class ConnectTest_MySQL{

public static void main(String[] args){

String url="jdbc:mysql://localhost/dataiso";

/* 连接到localhost上的bookshop数据库 */

String user="root";

String password="wwuhn";

String sql="INSERT INTO family(name,address,date)"+"VALUES('wwu','gz','2013/1/1')";

try{

Class.forName("com.mysql.jdbc.Driver");

Connection conn=DriverManager.getConnection(url,user,password);

System.out.println("conncting databse successfrlly!");

Statement stmt=conn.createStatement();

int rowCount=stmt.executeUpdate(sql);

System.out.println("成功新增"+rowCount+"笔数据！！");

conn.close();

}catch(ClassNotFoundException e){System.out.println(e);}

catch(SQLException e){System.out.println(e);}

}

}

创建对象最常用的方法就是用new运算符和类的构造方法，实际上也可以使用Class对象得到一个实例；

Class forName(ClassName);有一个newInstance()方法；
