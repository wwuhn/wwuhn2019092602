进程相关

进程system:windows系统进程，一组系统底层服务线程的总称，没有真正意义上的映像；

system idle process:用于显示CPU可用资源的百分比情况。该进程是作为单线程运行的，并在系统不处理其他线程的时候分派处理器的时间。

9个最基本的系统进程
必须加载,否则系统无法正常运行;
smss.exe window session manager process
csrss.exe client/server runtime process
winlogon windows logon process管理用户登录和退出
services windows services controller
lsass local security authority process
svchost service host process
spoolsv printer spooler service
explorer windows explorer
system idle process percentage of time the processor is idle

csrss.exe:windows环境子系统的进程映像，用于启动用户的会话模式。

Isass.exe:安全机制管理进程，用于管理IP安全和登录策略等。

explorer.exe:桌面环境进程，用于显示系统桌面上的图标及任何栏等。

winlogon.exe:用户登录管理进程，负责交互式登录、注销与关闭任务。

smss.exe:会话管理子系统，负责会话环境的初始化操作。

svchost.exe:系统服务的宿主进程，如果有多个svchost.exe同时运行，表明当前有多组服务处理活动状态或多个系统文件正在调用它；

services.exe:系统服务管理进程

进程具有各种状态

1 正在执行的状态；

2 等待执行的状态；

3 休眠状态；

4 僵死状态；

系统进程

1 空闲Idle进程

2 System进程；
