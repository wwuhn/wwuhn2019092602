帐户基础

使用一个帐户就像作用一台独享的计算机，每位用户都有属于自己的该当，图片，音乐和视频文件夹，用于存储相关文件，每个用户帐户可以拥有自己的电子邮件帐户和Internet收藏夹，每位用户都可以按自己的喜好自定义桌面，开始菜单和其它设置；这些内容都是用文件夹（系统盘下有一个用户users的文件夹）和注册表（HKEY_CURRENT_USER、HKEY_、USER）对应的；

桌面desktop:对应放在桌面的文件夹和文件；

收藏夹favorite:对应IE收藏的网址；

链接Link：对应资源管理器收藏的文件或文件夹快捷方式；

我的文档、下载、链接；

删除一个有人使用过的帐户时一定要小心，因为同时会删除它所有的电子邮件，电帐贴和internet收藏夹，如果不小心，还有可能删除该用户文档所有文件，包括我的文档和在桌面保存的文件；

删除一个帐户：会询问，是删除文件还是保留文件，但无法保存帐户的电子邮件和其它设置；（如需保存，可以使用windows轻松传送）

WWU使用一段时间后，AppData的数据量会骤增；

当删除一个帐户、新建一个帐户，需要做以下工作：

输入法需添加；

网络宽带连接需要重新建立；

IE需要重新设置，当然，收藏夹会没了，如果有备份，可复制到新帐户的上述路径；

桌面、任务栏需重新设置；

将帐户的全部设置轻松保存到一个系统文档利用轻松传送功能；

用net user命令进行帐户管理

黑客入侵成功后，一般都会自己添加一个自己的帐户，并将其加入到管理员用户组。

1 添加新帐户：net user hacker 123 /add

2 把帐户添加进系统管理员组：net localgroup administrator hacker /add

3 禁用系统帐户：net user hacker /active:no

4 激活已禁用帐户：net user hacker /active:yes

5 删除一个帐户：net user hacker /del

添加用户帐户管理的功能非常有限，（只区分标准用户和管理员用户同）

用户组是用户帐户的的集合，具有相同的安全策略。

本地用户和组：进行用户和用户组的管理和查看

计算机---管理---本地用户和组

每一个用户都隶属于某一个组。

用户可以新建，可以删除或禁用，可以将用户添加到不同的组；

组可以新建，可以为组赋予不同的权限：控制面板---管理工具---本地安全策略---本地策略---用户权限分配（双击，可以看到该权限分配给了哪些用户。

帐户类型：system帐户、管理员帐户、标准用户帐户

 

禁止特定软件在另外一个帐户中运行

win+R---secpol.msc：打开“本地安全策略控制台”---软件限制策略单击---右边窗口：其他规则右击，新建路径规则---路径：指定特定软件安装路径的exe文件，安全级别：不允许；

 

ABC帐户只能在规定时间内登录

win+R---CMD---net user abc 123 /add /times:Monday-Friday,19:00-21:00;sa-su,18:00-22:00

net user test /time:m,9am-6pm; t-f,8:00-17:00 /add

 

忘记自动登录密码

CMD---net user wwu 1234

CMD---以管理身份运行；

新建Dword32键值，键名：Administrator,值为1；

CMD:net user administrator /active:yes,然后回车；
