高级启动
当我们打开计算机并在Windows启动之前按 F8 键，都会进入“高级引导选项”屏幕，使您能够以高级疑难解答模式启动Windows，从而可以有效解决在正常启动进入Windows而不能解决的问题。
修复PC: 显示可以用于修复启动问题的系统恢复工具的列表，运行诊断或恢复系统。此选项仅在计算机硬盘上安装了这些工具之后才可用。如果具有 Windows 安装光盘，则系统恢复工具位于该光盘上。
安全模式: 以一组最少的驱动程序和服务启动 Windows。可查看加载的驱动和系统服务；local machine\system\current controlset\control\safeboot；
网络安全模式： 在安全模式下启动 Windows，包括访问 Internet 或网络上的其他计算机所需的网络驱动程序和服务。
命令提示符下的安全模式 少了一个网卡驱动，将GUI更改为命令提示符界面；使用安全模式下的命令提示符窗口启动 Windows，而不是通过一般的 Windows 界面启动。此选项适用于 IT 专业人士和管理员。
启用系统日志： 正常启动并创建了一个Ntbtlog.txt的日志文件；（包括正常启动时加载的驱动，和无法正常启动的驱动）；创建文件 ntbtlog.txt，该文件列出所有在启动过程安装并可能对高级疑难解答非常有用的驱动程序。
启用低分辨率
（640*480）		 使用当前视频驱动程序和低分辨率及刷新率设置启动 Windows。可以使用此模式重置显示设置。
最近一次的正常配置 使用最后一次正常运行的注册表和驱动程序配置启动 Windows。
目录服务还原模式（服务器） 启动运行 Active Directory 的 Windows 域控制器，以便可以还原目录服务。此选项适用于 IT 专业人士和管理员。
调试模式 以供 IT 专业人员和系统管理员使用的高级疑难解答模式启动 Windows。
禁用系统失败时自动重启 因错误导致 Windows 失败时，阻止 Windows 自动重新启动。仅当 Windows 陷入循环状态时，即 Windows 启动失败，重新启动后再次失败，使用此选项。
禁用强制驱动程序签名 允许安装包含了不恰当签名的驱动程序。

按下主机电源，电源就开始向主板和其他设备供电，在开机瞬间，电压不是很稳定，主板上的芯片组会向CPU发出并保持一个Reset信号，让CPU内部自动恢复到初始状态，当芯片组检测到电源开始稳定供电时，便撤出重置信号，CPU马上从FFFFFOH处执行指令，准备执行系统BIOS程序；

系统BIOS开始加电自检，并调用初始化代码来初始化配置相关设备；

系统BIOS查找显卡BIOS，屏幕会有一些初始化的信息显示；

查找完其它设备的BIOS后，系统BIOS将显示出它自己的启动画面，包括系统的BIOS类型、序列号和版本号；

硬件检测；

读取CMOS设置，以便知道有哪些设备可以启动？BIOS根据用户指定的启动顺序从软盘、硬盘、或光驱启动；（假设设置为从硬盘启动）

系统BIOS载入硬盘的MBR分区；

MBR的引导程序查找活动分区的引导扇区的地址；

引导权交给这个引导扇区去引导操作系统；载入windows XP中的启动文件NTLDR；

操作环境由DOS真实模式转换到32位操作模式；NTLDR会去寻找电脑中所有的硬盘，以便知道一共安装有多少个操作系统，并分析Boot.ini文件中所记录的操作系统信息，如果有安装多个操作系统，就会显示Boot.ini文件中设置的启动菜单，让你选择，然后系统检查硬件并利用硬件配置文件来设置各项硬件；

NTLDR载入windows XP的核心文件NTOKRNL.exe、各项驱动程序、服务，在这个阶段中，windows会载入注册表与系统有关的项目，并将之交给图形界面控制；

操作系统核心开始初始化，载入各种硬件信息，并由session manager启动windows XP的各项服务、输入输出设备。接下来，windows系统即启动成功；

启动的实质:

运行配置:将软驱或光驱或U盘上的dos的config.bat或光盘上的PE或安装在硬盘上的操作系统的配置文件调入内存运行;

按配置运行相关服务和驱动:将软驱或光驱或U盘上的dos或光盘上的PE或安装在硬盘上的OS的驱动或服务调入并常驻内存;

将相关程序如explorer.exe运行并驻入内存;
