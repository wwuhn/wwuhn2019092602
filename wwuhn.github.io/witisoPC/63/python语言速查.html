  2. 标识符和关键字

    3. 标识符是程序里的名字，用于命名变量、函数、类型等。常用的标识符形式为字母开头的字母、数字序列，下划线视为字母；

    大小写敏感；

  3. 代码结构

    4. Python程序以行作为代码单元，一行结束（到换行符）表示一个代码单元或结构成分（如表达式、语句、语句头部等）结束。如果在一行最后出现续行符\，下一行的内容就是本行的继续（相当于同时删去红续行符和换行符）。

    在python解释器中输入循环时，循环体语句如for i in range(0,50,10):在输完:后回车，需要输入四个空格作为缩进（不要使用tab），表示循环体的输入，循环体全部输入完成后，双回车表示循环体输入完成。

  6. 运算符和表达式

    7. 表达式基于字面量、变量等与运算符构成。不同类型的对象可使用的运算符不同。

    基本表达式包括各种原子表达式（字面量、变量等），用圆点形式描述的属性引用，用方括号描述的下标和切片表达式，用圆括号描述的函数调用式。

    运算符的优先级决定表达式的求值顺序。如果表达式中出现相邻的优先级相同的运算符，乘方运算符从右向左结合，其他运算符和（可相邻出现的）其他运算符都从左向右结合。另外，一元运算符总是从右到左逐个使用。

  7. 语句

    8. 如果一个语句（或语句的一部分，例如if语句的else部分)包含头部和语句组，当语句组是一个简单语句时可以放在同一行（头部之后）。任何情况下（包含语句组是一个简单语句的情况），都可以把语句组写在头部换行之后，以退格方式写在随后的行里。如果一个语句组里包含多个语句，它们的开始位置必须相互对齐。
