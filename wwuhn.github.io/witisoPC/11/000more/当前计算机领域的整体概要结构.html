人类机械物理工具介入人类智能活动的第一步，也是最基本的一步始终是有限字长二进制数字的基本计算或逻辑运算，以及对它们的存储。

真正的电子器件做计算机的开拓过程，经历了从制作部件到整机，从专用机到通用机，从“外加式程序”到“存储程序”的演变。

当前计算机产业分为两个互相联系而又相对独立的核心过程：通用计算机产品的制造过程与应用软件的开发过程。操作系统等软件可以看作是应用软件过程的一个特殊产物。这两条线的结合便是最终的计算机应用系统。

计算机的应用，粗略可以分为两大领域方向。一个是信息处理，一个是过程控制。

所谓信息处理，指计算机（及相关设备）作为一个独立完整的人工系统，人们输入信息（数据），计算机系统进行信息处理并输出信息（数据）。

数据结构、数据库的数据处理：增查删改；

而在过程控制类应用中，计算机自身不是一个完全独立的系统，而是一个更大的人工系统中的一部分，计算机信息（数据）承担处理任务，其输出的信息（数据）用于系统的控制，而不是提供给人使用。

业务模型的抽象，通常包括三个互相关联的部分：业务数据/信息模型；业务功能结构模型；业务流程/过程模型。前者描述数据/信息的构成与关系，后两者结合在一起描述完整的业务活动。

 

对图灵机原型的直观理解很难，因为一次只能读写0 和 1（一般计算机能读写64位），只能左右跳一步（一般计算机根据地址随便跳），比直接用机器码编程还要难，所以不要纠结了。

但图灵机引入的概念其实并不复杂，相当于一个有限状态机 + Memory, 状态机不难理解，直观一点比如不太贵的计算器，其实图灵机以前，帕斯卡，莱布尼兹，搞得都是计算器，叫的逼格高一点就是有限状态机。有限状态机加上 无限长磁带（其实就是内存的概念）就是图灵机了。而状态机其实又相当于图灵机只能读，而且只能在磁带上往一个方向走。 内存又其实是把状态机的可能状态变成无限了。
