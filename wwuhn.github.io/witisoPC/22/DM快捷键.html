Dreamweaver快捷键(英文输入法下使用)
shortcut function 适用在代码下 适应在设计下
shift+enter 换行<br />    
ctrl+b <weight></weight>    
ctrl+shift+p <p></p>    
ctrl+L 创建超链接   打开对话框
ctrl+shift+L 删除超链接    
ctrl+alt+i 插入图像   对开对话框
ctrl+j 页面属性    
ctrl+y 重复    
ctrl+shift+space 不换行空格    
ctrl+m 表格插入行    
ctrl+shift+a 表格插入列    
ctrl+shift+m 删除行    
ctrl+shift+- 删除列    
ctrl+alt+m 合并单元格    
ctrl+alt+s 拆分单元格    
Ctrl+T 打开快速标签编辑器　   设计模式下
