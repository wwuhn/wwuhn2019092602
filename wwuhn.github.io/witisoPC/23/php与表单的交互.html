表单页:.html

<form name="form1" method="post" action="dealwith.php"》

label：for属性与控件元素的id属性匹配；

submit：数据提交的触发；

reset

method:指定表单数据的传送方法，有get和post两种方法；

1 get方法：表单数据当做URL的一部分传送过去，http://url?name1=value1&name2=value2…;PHP使用预定义数组变量$_GET[name],也可以写成$name(需要在php.ini配置文本中进行预设置）；但url有长度的限制（<1024k)

2 post方法：提交的数据不依赖url，在后台传输，使用预定义数组$_POST[name]来获取表单的值；

action属性：表单数据所请求的url地址；

接收页：.php

复选框：checkbox

name的名字可以是一个数组形式，如name="chkbox[]",在返回界面可以使用count()函数来来计算数组的大小，配合for循环输出；

多选框

name属性应设置为数组；当PHP引擎发现在提交的字段名之后有一对方括号时，会给相应的超全局数组中的相应元素创建一个嵌套的数组，而不是创建一个单一的值，（对于多选列表框、复选框，未选中的框或按钮的数据不会做为元素发送；

表单的每个控件都有一个name属性，它表示用来存储数据的字段名，value对应的是字段值；

$_GET数组：包含所有用GET方法发送的字段名和字段值；

$_POST数组：包含所有用POST方法发送的字段名和字段值；

$_REQUEST数组：是以上两个数组+$_COOKIE数组的组合；
