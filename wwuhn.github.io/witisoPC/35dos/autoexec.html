autoexec.bat是一个特殊的批处理程序，它在每次启动计算机时自动运行。Autoexec.bat文件可以含有任何想在系统启动时执行的命令。如，自动清除系统中的临时文件，清除屏幕上的启动信息，或者运行用户喜欢的程序.放在启动盘根目录下.右击---编辑---可用记事本打开
@echo off 不显示命令行
prompt $p$g 设置提示符前有目录提示；
path c:\dos;c:\;c:windows;c:\tools; 设置路径；
lh c:dos\dosdey.com 加载键盘管理；
lh c:mouse\mouse.com 加鼠标管理；
lh c:dos\smartdrv.exe 加载磁盘加速管理；
lh c:dos\mscdex /s /d:mscd000 /m:12 /v 加载CD-ROM驱动；
set temp=c:\temp 设置临时目录；
echo 显示此命令后的字符；
echo off 不显示此命令后的所有命令的命令行本身
@ @:与echo off相像，但它是加在其他命令行的最前面，表示运行时不显示命令行本身。
lh or loadhigh 表示将程序加载到高端内存；
call 调用另一条批处理命令；
pause pause:运行此句会暂停，显示press any key to continue,等待用户按任意键后继续。
rem rem:表示此命令后的字符为解释行，不执行。
