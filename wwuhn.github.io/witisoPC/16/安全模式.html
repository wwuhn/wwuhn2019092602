• 安全模式：由最小集合的设备驱动程序和服务构成的一种引导配置。避免加载那些可能会崩溃的第三方和其他非关键的驱动程序。

    • 安全模式下磁盘查错和程序运行

    进入带命令行的安全模式；

    进入C:\windows\system32并在\>chkdsk d: /f /r;(C盘需在设置为重启后查错）；(chkdsk可在任意目录下运行，内部命令可在目录下运行？

    在命令行模式下查完D：盘，再查F盘时，按向上箭头，命令提示为上一个命令，再将d:改为F:即可；

    在命令行模式下运行程序，需要进行到该目录下才可以运行，如运行浏览器：

    c:---dir---cd program files---cd internet explorer---iexplore

    power后，按F8选择进入

    安全模式：使用一组最少的驱动程序；

    较少的驱动、较少的硬件；较少的服务、较少的功能；
