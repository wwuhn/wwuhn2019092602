三款主流数据备份软件评测

对一些计算机用户而言，硬盘中存储的数据才是最为宝贵的资源，其无形价值甚至高于电脑本身。与其在数据丢失时费时费力找人恢复，不如事先将重要数据备份出来以备后患。下面就向大家介绍几款数据备份软件。

测试平台

CPU：P4 1.8GA

主板：Intel原厂845G

内存：Kingston 256MBDDR

硬盘：酷鱼Ⅳ 80G

操作系统：Microsoft Windows XP Professional

参评软件

1、腾龙备份大师 2004 黄金企业版 v6.4 Build 80

2、Acronis True Image v8.0 Build 786

3、Norton Ghost v9.0.0

从外观界面而言，“腾龙备份大师”最显时尚化，而Acronis True Image与Norton Ghost的界面布局则略显雷同，设计较为简洁、传统。

一、易用性

这三款参评软件均采用了向导式的操作方式，用户只需亦步亦趋地按需设置即可，显得非常规范化。Acronis True Image作为其中唯一英文界面的软件，其易用性无疑会打些折扣。

三、备份介质支持

1、腾龙备份大师

程序支持本地及网络硬盘、移动存储设备及磁带机（特殊版本）等Windows可以识别的驱动器。

2、Acronis True Image

除了常规的各种存储设备之外，还支持硬盘对硬盘的克隆。

3、Norton Ghost

支持的备份介质与Acronis True Image近乎一致。

五、实时监控备份数据

实时监控备份目标（包括网络监控）的变化，进而及时更新数据，是“腾龙备份大师”的优势功能。

实时监控备份目标

这种同步功能的使用，使得数据的备份更为智能化了，不过，若当前备份目标是频繁增加、删除文件，则定然会在一定程度上影响到系统运行速度。

Acronis True Image与Norton Ghost均不支持实时监控。

六、检测映像文件

1、腾龙备份大师

利用程序的项目检测系统，能够检测出备份项目的原始文件或目录丢失。

2、Acronis True Image

用户可选择在还原前自动检测备份文件或是随时通过程序命令进行检测。

检测备份文件

3、Norton Ghost

用户在执行还原操作时，其默认设置为验证镜像文件，以检测是否有文件丢失或错误。

验证镜像文件

七、压缩备份文件

1、腾龙备份大师

程序支持快速、默认、最大三种压缩方式。

2、Acronis True Image

程序支持不压缩、正常压缩、高度压缩及最大压缩率四种方式。

3、Norton Ghost

程序支持不支持、标准、中、高四种压缩模式。

八、映像文件保护

1、腾龙备份大师

如果选择备份类型为自动锁定，可配合杀毒软件与防火墙等安全防护工具为数据提供全方位的保护。

2、Acronis True Image

我们可为镜像文件设置密码保护。

3、Norton Ghost

虽然程序同时支持为备份文件设置密码，但其选项设置为较为隐蔽，初使用者不易于看到。

九、高级备份方式

1、腾龙备份大师

支持增量备份及差分备份。

2、Acronis True Image

不支持此类备份方式。

3、Norton Ghost

支持增量备份。

十、切分备份文件

1、腾龙备份大师

不支持此项功能。

2、Acronis True Image

在备份时，用户可自行选择镜像文件大小，支持由系统根据目标空间的大小自动进行分割或直接选择“1.39-3.5’’Floppy”软盘、“100 MB—ZIP-100 Disk Drive”、650 BM/700 BM的刻录光盘分割镜像文件等容量。

3、Norton Ghost

在驱动器备份向导中，用户可根据程序预设的文件大小选择切分备份镜像。

十一、计划任务

1、腾龙备份大师

在计划数据备份方案中，用户可选择分钟、小时、每周、季度、月份及年度的备份方案。

计划任务

2、Acronis True Image

用户可选择在每日、每周、每月、指定时间点、计算机启动、计算机关闭或一天N次等情况下有计划地执行数据备份操作。

任务向导

3、Norton Ghost

用户可选择在指定时间点创建备份或每周、每月执行备份任务。

备份任务向导

总结

这三款数据备份软件各具特色，“腾龙备份大师”虽名气不敌其它两款软件，但在某些方面的确具有一定的优势。但是，在硬碰硬的数据备份速度的较量中，其劣势明显。而且在还原数据时，偶尔还会出现错误令程序假死。看来该软件在自身性能上还有待提高。

而Acronis True Image与Norton Ghost相比差异不大，Norton Ghost只是略占些优势而已。对于很多国内用户而言，从界面友好性及操作效率等方面考虑，还是使用Norton Ghost更为稳妥。
