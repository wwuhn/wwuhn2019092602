四 程序安装

  1. 设置一个系统还原点
  2. 程序安装目录更改（最好不要改，会有很多问题），另外如果需要做gho的话，系统与程序最好安装到一起；
    1. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion]:ProgramFilesDir
    2. 把C:\Program Files\Windows Photo Viewer放到更改后的目录,不然看图会有问题;
  3. 注意事项
    1. 一些阉割过的版本会有诸多问题：一些功能用不了，兼容不好；用过一个改动过的会声会影，无法使用分享功能；
    2. 越大、功能越复杂的程序安装时的要求越高，时间越长，很容易安装失败，需要较优的安装环境；
    3. 安装时要注意一些开机自运行、快捷方式、关联文件、默认勾选的一些工具的选项选择；尽量选择自定义安装（自定义安装位置、组件等）；
    4. 一些程序要求先安装一些支持部件或公用程序：.net framework v4.0.30319\microsoft visual c++ 2010;
    5. 一些程序需要重启一次继续安装，如nero;一些程序需要重启后才可运行，如symantec norton ghost 14;
    6. key:Ⅰ 一些程序要求先输入key,才可开始安装，如CAD;Ⅱ 一些需要安装完成后，需要输入key才可运行，如ps;
    7. Ⅲ 一些程序可以让用户试用一段时间，如ultraiso9;
    8. 最新版本的程序不一定是最佳的选择：bug、兼容不好、缺少技术支持；
