电子显微镜下的CPU
对于CPU的核心部分，通常我们只在幻灯片中才能见到示意图，很少有人亲自在微观尺度下检验过到底是什么样子。而这位小学/初中教科学课的瑞典教师为了教学需要，借来了一颗Pentium III 800EB Coppermine CPU，用电锯拆解后放到电子显微镜下拍摄了一组照片，虽然他用的普通电镜分辨率最大只有2000nm，没有移除金属层还不足以看到晶体管的细节，不过即使是在10000nm分辨率的等级也足以让我们看清平常天天打交道的CPU核心部分。
