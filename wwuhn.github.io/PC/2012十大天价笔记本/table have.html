随着2010年和2011年对平板市场的全情投入收效甚低，因此在过去的这一年中，对于OEM厂商来说已经将主要注意力放到了刚刚起势的超极本上，以至于我们看到去年笔记本市场新品的数量要大幅低于前年，而厂商转移注意力更深层次的原因则是由于传统笔记本过低的利润率，要知道作为全新品类的超极本在当前贡献的利润率要远高于前者。

由于被超极本、平板电脑双重夹击，过去一年价格战可谓是更加迅猛，毫不夸张的说，如今一台iPad的价格就能买到一台中端性能的笔记本，以至于很多人会认为高端市场将愈发走向真空的状态。但实际情况绝非如此，据我们了解，一台高达三万元的Alienware高端娱乐本，在暑促期间每天都可以出货两到三台，并不亚于那些高性价比机型。当然，如果从价格角度作为考量标准，三万元也仅仅是处在中档的标准，下面我们就来对过去一年那些天价笔记本做一个盘点。
