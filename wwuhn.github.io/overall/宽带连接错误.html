Error 602 The port is already open

　　 A：拨号网络由于设备安装错误或正在使用，不能进行连接。原因是RasPPPoE没有完全或正确地安装，卸载PPPoE软件，重新安装即可。

　　 Error 605 Cannot set port information

　　 A：拨号网络由于设备安装错误不能设定使用端口。原因是RasPPPoE没有完全或正确地安装，卸载PPPoE软件，重新安装或更换PPPoE软件即可。

　　 Error 606 The port is not connected

　　 A：拨号网络不能连接所需的设备端口。原因是RasPPPoE没有完全或正确地安装，连接线故障或网络连接设备出现故障。卸载PPPoE软件重新安装，并检查网线和ADSL MODEM。

　　 Error 608 The device does not exist

　　 A：拨号网络连接的设备不存在。原因是RasPPPoE没有完全或正确地安装，最好卸载PPPoE软件，重新安装。如果仍无法解决，可以更换其它拨号程序来进行拨号。

　　 Error 609 The device type does not exist

　　 A：拨号网络连接设备的类型不能确定。原因是RasPPPoE没有完全或正确地安装，卸载干净任何PPPoE软件，重新安装该软件或者选择其他的软件。

　　 Error 611 The route is not available/612 The route is not allocated

　　 A：拨号网络连接路由不正确。原因是RasPPPoE没有正确地安装，或者ISP服务器出现了故障。解决方法是卸载系统中所有的PPPoE软件重新安装，如果问题依旧，可以询问ISP。

　　 Error 621 Cannot open the phone book file
　　 Error 622 Cannot load the phone book file
　　 Error 623 Cannot find the phone book entry
　　 Error 624 Cannot write the phone book file
　　 Error 625 Invalid information found in the phone book
　　 A：以上都是 Windows NT 或者 Windows 2000 Server 网络RAS网络组件出现故障。此时应卸载所有PPPoE软件，并重新安装RAS网络组件和RasPPPoE

　　 Error 630

　　 A：ADSL MODEM没有响应。原因是连接ADSL MODEM的电话线出现故障，或ADSL MODEM工作不正常，如电源没打开。此时应重点检查ADSL
　　 设备的工作情况。

　　 Error 633

　　 A：拨号网络由于设备安装错误或正在使用，不能进行连接。原因是RasPPPoE没有正确安装在操作系统中，导致网络连接错误。请卸载电脑中的PPPoE软件，然后对其重新安装。

　　 Error 638

　　 A：无法连接到ISP的ADSL接入服务器。原因可能是ISP端服务器出现故障；或者在RasPPPoE所创建的拨号连接中错误地输入了电话号码解决的方法是使用拨号程序RasPPPoE.exe，检查其是否能列出ISP服务，以确定ISP端的工作情况；如果是电话号码输入错误，则把所使用的电话拨号连接中的号码重新输入过。

　　 Error 645

　　 A：网卡没有正确响应。原因是电脑中的网卡出现故障，或者网卡驱动程序没有正确安装。此时就须要检查电脑中的网卡是否出现硬件故障，如果没有，则应重新安装网卡驱动程序。

　　 Error 650

　　 A：远程计算机没有响应，断开连接。原因是ADSL ISP服务器端出现故障，或网卡有故障，也可能是网络协议出现错误。解决的方法是检查ADSL MODEM的信号灯是否能正确同步，检查网卡的工作情况，或删除所有网络组件，重新安装网络协议，再次对计算机进行配置。

　　 Error 797

　　 A：ADSL MODEM连接设备没有找到。可能是ADSL MODEM的电源开关没有打开，或者是网卡与ADSL MODEM间的连接线出现问题。也可能软件安装以后，相应的协议没有与硬件设备正确地绑定，在创立拨号连接时，建立了错误的空连接。

检查电源、连接线是否有问题；

检查网络属性，主要是与RasPPPoE相关的协议是否正确地安装并绑定，在“设备管理器”中检查网卡是否出现问号或感叹号；

检查拨号连接的属性，是否连接的设备使用了一个“ISDN channel-Adapter Name(**)”的设备，该设备为一个空设备，如果使用了，请取消它，并选择择一个正确的PPPoE设备来代替它，或者重新创立拨号连接。
