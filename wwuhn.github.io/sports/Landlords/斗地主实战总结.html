1. 开局时在记牌的同时，要分析另两家叫牌及可能的牌好牌差的情况，结合自己牌面的情况，明确自己的角色；
  2. 要精细，如有一定要出的三还一的牌，过小牌时，可考虑留一个最小的单张给三还一；
  3. 终局时，地主是单张，或多张单张时，顶家和下有的策略都是顶大发双；
  4. 下家不要死盯着过小张，下家也要注意顶地主的牌；
  5. 炸弹该拆时要拆，该拖二时要拖二；如果是顺子中张的炸，（自己是农民时）可考虑保留这一牌型到终局，看盟家是否能报单？
  6. 要不用力过猛，一是会过早暴露，二是牌型会少了变化和应对；

牌型举例

  1. 一般情况D不压2，如果盟家有X，会压，如果没有而在地主家，D可等着压地玉的X；
  2. 2X+一小对，考虑先出2；
  3. 2+单张，也要考虑先出2,报单的牌容易被控制；

失败举例

  1. 一炸+1单，老盯等着过单，结果输；
  2. 3455678899SSJJQQKADX,上家还有7张版，上家打J时，没有跟Q，盲目开炸；
