腰椎间盘突出症的自我疗法

1、生活习惯：在睡觉时，弯曲髋关节横向侧卧，或者在腿下面垫上垫子屈腿仰卧。枕头要用稍硬一些偏低的;睡醒后应先床被上将腿屈起做左右倒的体操，然后再用胳膊支撑上身起床;保持正确的姿势坐在椅子上，腰间盘突出患者时而进行腰腿部及脚底穴位的按摩，或做一下体操以缓解腰部肌肉的紧张。
 

2、反复搓腰：将双手分别放于同侧腰大肌处，由上向下，再自下而上反复搓 10-15次，以双侧腰部发热为度。
 

3、爬行训练：四肢呈爬行状，先后做弓腰、沉腰动作。然后侧身左 (右 )手摸左 (右 )足，扬手转身等动作，最后将双手着地，作爬行动作，每天坚持 30分钟。
 

4、飞燕式锻炼：俯卧于床，先后做双下肢交替抬举、双下肢同时抬举、上半身后伸抬起、身体两端同时抬离于床等动作，上述动作各十余次，每日坚持 30分钟锻炼。
 

5、团身运动：仰卧于床，先后做屈髋、屈膝、仰卧起坐或仰卧起坐接双手抱膝贴胸等动作各十余次。此运动可与飞燕式隔日交替进行。
 

6、退走锻炼：在走廊或空旷处倒退走，每次 30分钟左右。这种锻炼有利于改善腰背肌状态，恢复腰椎生理弓和腰部小关节滑动，可帮助解除小关节粘连。
 

腰椎间盘突出不是小问题，一定要去医院确诊，并及早治疗。上面的锻炼方法只能起到一定的辅助作用，不能从根本上治愈疾病，希望能引起患者的重视，以免耽误你的治疗。
