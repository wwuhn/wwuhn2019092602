运输方案设计管理

  1. 运输方式选择
    1. 调查并掌握铁路、公路、水路、航空运输等各种运输方式的优缺点及适用范围；
    2. 用《输运方式选择表》对每种运输方式的评价指标进行打分，然后选择分值最高的运输方式；
    3. 当单一运输方式不能满足运输需求或者成本过高时，调度人员应当采用多多联运的方式完成运输；
  2. 运输路线选择
    1. 熟知运输网络节点路的远近、不同运输工具成本的高低、不同时间段线路的容载情况、不同线路的时速限制以及车辆的正常行驶速度等，然后根据成本较低和速度较快的原则确定议案；
    2. 从同一发货点运输到不同收货点时，应根据实际问题建立模型，运用最短路径法选择路线；
  3. 承运商选择
    • 如果使用铁路、水路、航空运输或者公司公路运输动力不足时，调度人员应当选择合适的运输公司托运货物，选择承运商的标准和方法请参考本公司相关运输外包制度；
  4. 运输路线方案的制定与审批
    • 编制《运输路线规划方案》申请审批；
  5. 运输工具需求量计算
  6. 车辆安排
    1. 了解每台车辆的质量、性能和优势、吨位、容积；
    2. 选择合适吨位的车辆，严格禁止超载；
    3. 参考货物的尺寸和吨位，选择合适容积的车辆；
    4. 根据货物的包括和性质等，选择不同货厢形式的车辆；
    5. 为距离远、路况复杂的运输安排车况较好的车辆；
  7. 驾驶员安排
    1. 安排驾驶经验丰富、技术好的驾驶员执行难度大的运输任务；
    2. 安排维修技术水平高的驾驶员执行路况较差的长途运输任务；
    3. 安排态度好的驾驶员完成复杂或与客户接触较多的任务；
    4. 涉及到转关运输和保税物流或遇到业务单证多、环节多的情况，需要安排业务文化水平较高的驾驶员；
    5. 调度员需要清楚地了解货运驾驶员的身体状态和家庭情况，合理为其安排出车任务；
    6. 400公里以上的长途运输原则上应安排两位驾驶员，以确保行车安全。
  8. 运输过程调度管理
    1. 发运管理：填写《运输派车单》，录入物流信息系统，并交相关人员；
    2. 在途控制：对运输车辆进行跟踪，及时处理突发情况；调度人员应经常与押运人员进行电话沟通，及时掌握车辆运行情况，保证运输任务能及时完成。
    3. 到货安排：运输人员在到达目的地前，应提前通知到货站调度人员收货准备；并为运输车辆和人员安排好返程任务；
