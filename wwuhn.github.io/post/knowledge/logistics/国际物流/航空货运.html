航空货运

广义的航空货物运输可分为3种，即普通货物运输、邮件运输和快递运输；随着航空运输市场的发展和竞争，三者之间的差异越来越小；航空货物运输方式主要有班机运输、包机运输、集中托运、航空快递；

2 航空运费

2.1 起运费：类似TAXI的起步价，不同地区的起步价与TAXI一样不一样；

2.2 航空货物运费的计费重量

体积重：国际：7000cm3折算为1kg；国内：6000cm3折算为1kg;

(汽车：3.0m3折算为1T；1m3=333.3kg)

(铁路: 3.3m3折算为1T；1m3=300.0kg)

2.3 运费transportation charges：根据适用运价计得的发货人或收货人应当支付的每批货物的全部运输费用；

2.4 运价rates：是指机场与机场间的空中费用，不包括承运人、代理人、或机场收取的其他费用；运价主要分为四类：

2.4.1 普通货物运价general cargo rate,GCR：

a 45kg以下：比45kg以上费率要高；

b 45-300kg; "货物越重，费率越低，即费率由高到低；"

c 300-500kg;

d 500kg-1000kg以上；

e 1000kg以上；

2.4.2 特种货物运价special cargo rate或指定商品运价specific commodity rate CCR：在普通货物运价基础上计收，比如鲜活水产品(运价为普通货物运价的3倍)、液体货物等；

2.4.3 货物的等级运价cargo class rate CCR：是按照货物贵重程度不同而制订的费率，如精密设备或者珠宝等;

2.4.4 装箱货物运价Unitized consignments rate UCR,是以大宗货物以集装箱为单位进行空运时的价格；

2.4.5 优惠价； 

3 飞机类别

现在国际民航市场上共分窄体客机，半宽体客机，宽体客机以及超宽体客机。

3.1 非宽体飞机,不超过80kg;

窄体飞机：机身3m，旅客座位之间只有一个走廊,通常在其下货舱装散货，或有特的集装箱，高117cm,宽体的一般高163cm；

窄体客机是每排座位不超过六座的单走道客机；

3.2 宽体飞机,不超过250kg;

不超过100*100*140cm；宽体飞机有3排座位，二条走廊，宽4.72m；一般分主舱和下舱，波音747分上舱、主舱、下舱；（上舱在上，位置较少;

3.3 载重最大的民用飞机B47全货机：货物最大载重119吨，飞机飞行时速大约在600-800公里左右，现在的宽体飞机能飞7000km左右；

3.4 从中国飞到美国的西海岸大约需13小时，空运价格至少是海运价格的10倍以上；

空运禁运品名单

危险品：爆炸品、气体、易燃液体、易燃固体；

禁运品：粉末或液体状物品、药品、刀具、榴莲等；

空运方式

班机运输：固定航线，固定的始发和停靠港并定期开航，公司可以确切地掌握起运与到达的时间；

包机运输：包括整架包机和部分包机；

集中托运：将若干批单独发运的货物组成一整批向航空公司办理托运；

航空速递；

航空订舱：订舱人员依照发货人的要求和货物本身的特点选择最佳航线和最理想的承运人，同时争取最优惠、是合理的运价；订舱人员向航空公司领取并填写订舱单，写明货物的名称、体积、重量、件数、目的港及要求出运的时间等，民航公司根据实际情况安排航班和舱位；订舱人员按货运单接收货位到周转仓库（急件和特件除外），等待配仓人员指示；空运货物入库完毕后，配舱人员应核对货物的实际件数、重量、体积与托运书上填报的内容是否一致。配舱人员应根据货物特点有效、合理地拱配预定仓位、板箱。配舱人员对晚到、未到以及未能通关的货物作出调整处理，为配舱做准备；

在货物发运后，订舱人员应及时将发运信息传递给客户，向其提供航班号、运单号和出运日期等；
