1 工作职责范围：

1.1 根据《每天出货计划》，综合客户需求、进行路线优化组合，车辆调度，编制《派车单》；

1.2 每日分拣和装车工作的统筹和安排，包括人员分工、相关物流设备的统筹和工作进度的控制，以及相关异常问题的沟通和协调；

1.3 建立车辆的保养制度和保养计划，以及每台车辆的费用核算和控制；

1.4 车辆相关资料和客户资料的建立和完善，包括不同客户、不同类别的货物所要求的装卸、搬运、保管、包装和运输等方面的详细要求；

1.5 物流成本和物流服务两方面的协调管理；

1.6 相关工作标准、工作流程、工作制度的建立和完善，并确保有效实施；

1.7 一套合理的异常问题沟通协调机制和持续改善机制的建立、完善并确保有效运行；

2 工作核心

2.1 配送各环节相关问题的沟通协调，以确保成本控制和客户满意的有效实现；

2.2 确保收货、发货以及单据、相关数据、报表的规范、及时、准确；

2.3 确保库房每一区域规范、物流有序；

2.4 确保每天工作的质量和工作进度，以制度监控和人员的有效激励去减少各方面可能出现的差异；

3 工作目标

3.1 尽可能少的问题以符合相关方的期望（相关方包括公司内部的每一个部门以及公司外部的承运商、客户等）；

3.2通过物流功能的最佳组合，以有效的库容、运力利用、及时输送和快速反应客户需求来追求物流系统整体最优；

4 具体工作思路

4.1 通过相关工作标准、流程、制度和相关表单的建立和完善；确保各个环节的逐步规范化运作；

4.1.1 告诉员工该做什么？岗位职责；

4.1.2 告诉员工该怎么做？管理制度、工作程序、作业指导书、表格单据；

4.1.3 告诉员工应该做到什么程度？目标体系或KPI体系；

4.1.4 告诉员工他的工作绩效是怎样被衡量的？检查标准和考核体系；

4.2 每一岗位建立清晰的岗位职责、岗位目标和考核标准并定期实施，以确保每一岗位的工作质量和效率；

4.3创建一个有竞争、有协作的团队工作氛围，通过培训、激励让每一个员工都可以有能力、有承担地去完成他的工作，确保每一个员工良好的工作状态；

4.4 自己的每一项工作都要有进度计划，并做到清晰明了、疏而不漏，以确保每一项工作的进度和质量都准确无误；

4.5 建立和运行合理的问题纠正预防和工作持续改善机制，确保错误不叠加、不重复、可追溯，确保工作的逐步持续改善；

4.6 确保物流与信息流高度同步且能准确一致； 即要求实际的物流操作、对应的存卡、单据、电脑数据系统的记录四方面应能准确对应（也就是能通过规范的单据沟通物流与信息流且严格一致），以信息系统为支撑实现物流全过程的同步化管理，要求定期或不定期的检查、核对、盘点、总结，并落实相应的纠正和预防措施；

4.7以供应链和系统管理的方法追求相关方的共赢或持续满意，通过目标管理来确保成本和服务水平两方面的有效控制；

4.8 做好各项物流要素的管理：包括物流人员、物流设施、物流装备、物流技术等；

4.9 做好各项物流功能要素的管理：包括储存、运输、配送、信息加工等；

4.10 物流技术标准、管理标准、工作标准、信息标准等方面的逐步推进；

4.11物流管理应掌握的一些基础数据：商品或货物的形状、尺寸、数量（散货数、箱数、托盘数、入库量、出库量、库存类、客户数），重量；

心得

1 以规范化运作和持续改善为关注焦点

要提高规范化运作的水平重点在于保证制度的完整性、符合性、以及执行力；而提高执行力的关键在于对员工的激励和团队建设，增强员工的责任感和积极性；

持续改善考虑的一些方面主要有：

1 纠正预防措施控制程序；

2 PDCA循环；

3 5S管理；

4 效率，每一个环节的操作的方法没有最好，只有更好；优化没有终点；

 

岗位职责：

1 收货与发料管理

1.1 监督收货和发料工作完成的时效性和准确率，确保生产顺畅；

1.2 监督收发过程中的批次管理执行状况和单据完整率；

1.3 收集收发过程问题点和反馈，及时改善；

2 在库管理

2.1 针对不同原材料种类状况全面规划仓库空间和物流路线布局；

2.2 确保在库材料的数据准确，按月安排全盘和配合循环盘点，对盘点差异进行原因分析和改进；

2.3 按原材料不同储存要求区分储存管理，对即将超出保质期的原材料提前预警；

2.4 负责仓库物料的防火 防潮 防虫害 防盗等工作；

2.5 管理仓库相关设备新增 使用 维修和报废。

3 数据管理

3.1 对仓库相关工作进行数据统计；

3.2 建立完整的仓储操作报表，反应仓储作业情况。

4 仓储流程优化和效率提升

4.1 拟定和改善仓库相关的工作流程和管理制度；

4.2 根据实际情况不断优化部门工作和人力安排，提升工作效率。
