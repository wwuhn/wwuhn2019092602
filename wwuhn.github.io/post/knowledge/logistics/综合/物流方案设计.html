Logistics Project Design

from:《物流方案设计》王清斌，东北财经大学出版社

分类，按主体：

每一个物流方案在设计时所针对的对象不同，物流服务需求也千差万别，所以物流方案具有很强的针对性。但一般来说，物流方案者是为了提升物流动作水平，为实现物流系统的高效率、低成本的合理化物流服务目标而设计的。

  1. 企业物流方案
    1. 工业企业物流方案
    2. 商业企业物流方案
  2. 区域物流方案(具有公共服务性质),需要考虑充分利用公共的物流设置和资源；
  3. 物流园区物流方案
  4. 项目物流方案，是一种解决工程及建设项目、大件运输和大型展会等的物流组织与运作过程，通过计划及合理的资源配置，达到一定特定目的的综合性复杂的且有工程特性的物流活动。其最大的特点是只做一次，一般不重复，因此其风险较高，时效性较强，需要在限定的时间内完成。一般搬迁项目物流、应急物流和战时物流等也可以归结为项目物流；

概念

物流是包括运输、仓储、配送等功能环节的复杂系统，在具体物流活动的动作管理中，需要按照系统化的要求和目标，遵循系统内各个功能要素相互联系、相互作用、相互影响的特点和规律来具体实施，需要按照一定的方式方法来进行。

这种物流活动操作的方式方法和技术手段体现了物流系统在资源配置基础上的一种合理化组织。

发现物流管理过程中影响效率和效益的本质性问题，对现实物流系统进行理论分析，运用理论方法对物流系统进行优化和规划，并对结果进行分析和调整，创造性地提出解决问题的措施和方法，使之形成可操作的物流运作流程。

物流管理是对相关运输、仓储等活动的计划、执行和控制，具体的物流操作需要按照计划来执行，在执行中加以控制和调整。

物流方案是合理化物流系统的具体形式，是物流管理活动不可缺少的内容。

在经济活动中，人们开展的与经济有关的各种活动可以分为两类，一类是周而复始、持续不断的活动，组织利用这些活动来生产产品或提供服务，称之为“动作”(Operation);另一类是为完成某一独特的产品或服务所做的一次性、临时的活动，称之为“项目”(Project)。

运作是按照操作标准和固化流程进行，并有持续改善；

项目是一个特殊的将被完成的有限任务，要求对时间、费用、技术、风险以及实施计划进行论证。

物流方案是从事物流活动的物流项目和物流运作总称，是为解决现实中物流系统和物流管理活动中存在的问题，实现物流管理系统化和合理化的一系列的技术、方法、手段、措施等的集成，是一系列可操作的标准化业务流程（SOP）。

物流方案的内容

  1. 宗旨和目标
  2. 物流现状分析
  3. 物流运作模式与业务流程

    4. 物流作业系统是物流系统规划的核心。也就是物流总体运作模式和各功能环节的作业模式及其具体化的业务流程。需要需要对方案所设计的各功能环节做总体描述，阐述各功能环节之间的关系和影响；其次要对物流空间网格结构做出详细的描述并说明选择这种空间结构的依据；再次是分环节的物流运作模式的说明，如运输方案包括干线运输方式（铁路、水路等单一方式或组合运输模式）及具体操作方法，RDC或仓库等管理模式，配送作业模式以及物流系统辅助功能如流通加工、包装等环节的具体作业方式，并对物流运作系统的功能环节的作业模式形成具体化右操作的作业流程，以流程图的形式表现出来。除了核心业务环节如运输、仓储、配送等传统的物流环节及功能外，在公共性较强的物流系统规划设计中，这部分内容还应包括商务管理、物流金融及汽车集散分拨等新型物流服务功能以及与核心业务相关的辅助功能等内容。

    另外，这部分内容还应该包括在物流运作模式认证过程的详细描述，包括空间网络结构设计的依据，合理化运输模式及优化方法，RDC等节点的选址方法，配送路线优化、配载优化等配送计划的制订方式及决策方法，物流系统及功能环节的外包决策方法等内容。

    由于优化的目标不同，可以提出多种方案供选择；

    如方案设计的主体是第三方物流企业，在为客户设计的物流方案中对每个功能环节要给予详细的服务价格说明。第三方物流企业给客户设计物流方案，必须把自己企业的资质、物流资源、物流服务优势等在方案中介绍清楚，使客户对物流企业有深刻的认识。比如，已有的车队，仓库的类型、数量，可控的车队规模，整合社会物流资源的能力，物流服务的经验和已做过的成功案例等。

  4. 物流信息服务模式，物流信息系统是物流系统的神经中枢。
  5. 物流运营组织模式，物流方案最习题集是由人(组织)去实施，因此在方案中还需要提出为运作实施方案所需的管理模式和组织结构，以及各部分人员配置、工作职责及对人员素质要求等内容。这部分内容的主体是物流组织体系结构。
  6. 方案实施措施及建议，包括实施中的调整，以及实施所需的条件和资源。

物流方案的详细设计

首先是物流系统的总体规划、物流整体运作模式设计；

详细设计是让其成为一个操作性强的物流方案；

物流方案设计是一个边论证边设计的过程；

物流方案设计的内容

  1. 概念设计：包括宗旨和目标，总体设计思路及应用的主要技术方法。在第三方物流企业为客户设计物流解决方案的情况下，要提出具体的服务承诺及关键性的成本和效率指标。
  2. 总体物流模式设计：
  3. 各物流环节功能与流程设计：
    在总体方案目标下，设计优化运输网络结构，选择合适的运输方式组合，根据客户服务需求进行内外运输问题的决策及动迁模式、车辆等运输资源的配置决策和方案、运输和配送的分包商服务采购及监控方法的确定，车辆调度优化与方法、配送模式与配送方法的选择，运输与配送的成本计算以及效益分析等，在明确运输与配送的模式和方法的基础上，设计合理的业务流程。
    根据总体物流运作网络结构，选择RDC与仓库的地址和类型，新建仓库的结构设计，改造现有仓库的计划，库存划分及库位分配策略等的设计与决策，仓库设施、设备选择、存储策略的选择，入库管理、在库管理、出库管理的方法设计等，主要仓储业务环节的流程与单据设计；
  4. 物流信息系统设计：
  5. 物流运营组织结构设计：
    不同各类的物流方案，其运营主体、运营模式有所不同，需要根据具体的情况设计物流运营的组织体系结构，具体包括运营组织模式的设计，工作制度设计等内容；
  6. 其它：
    1. 如其他辅助功能环节的计划，如包装、流通加工等辅助功能设计；
    2. 物流基础设施如消防、照明、市政基础设施对接等；项目建设与运营的环境评价与环境保护措施方案等。
    3. 物流服务质量保证措施，如客户投诉处理方法、动行故障处理程序等；
    4. 对于第三方物流企业所设计的物流服务方案，还应包括物流资源整体介绍、物流服务详细报价、物流服务具体指标的承诺等内容；

物流运营管理模式选择

自营、外包、合作、委托管理等；哪些环节或部分要考虑外包；

物流园区的投资主体一般都带有政府的色彩，但其运营管理一般都需要企业化运作。

运输与配送方案设计

物流网络主要包括两种要素：节点和线路；

货品在线路上主要是通过各种运输方式，以合适的运载工具进行一定频率和指的运输，是运行的状态；

货品在节点内的活动包括简单的装卸、存储等作业活动，测验为更合理地进行货品的分拔运输与配送而进行的批量转换、运输方式衔接、拣选配载、车辆配载、末端协同送货等相对复杂的作业活动，还包括物流增值的活动；

合理的网络结构可以提高运输效率，降低运输成本，重点要解决以下几个方面的问题

  1. 计划区域内建立网络节点的数量、位置、规模，及层级关系；
  2. 各路线上运输方式与运载工具的选择；
  3. 节点的进化与供货关系（客户与供应商的关系）；
  4. 物流服务质量水平以及信息网络模式；

仓储方案设计

  1. 仓储规划：仓库选址、存储策略选择、库区资源分配、内部仓储作业各类设备的配置、仓储设施内部布局、仓储作业整体模式等；
  2. 仓储作业管理：入库、在库、出库等作业活动的优化与流程设计，以及保证作业活动顺畅进行的各种仓储规章制度的制度等；
  3. 库存控制：是指本着经济的原则对各类库存品在满足一定的供应服务率的条件下的合理数量控制；
