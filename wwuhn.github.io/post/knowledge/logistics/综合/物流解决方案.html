1. 欧梵的物流问题
    1. 产品经销商对欧梵的忠诚度不够；
    2. 欧梵的产品物流系统较为混乱，采购、生产和销售难以实现一体化运作，无法为客户提供更优越的物流服务，没有充分利用已经建立的物流渠道；
    3. 销售网络从广度来看覆盖面不够，较为狭窄；从深度来看渗透力不够，只涉及一级代理商，对终端客户没有形成控制力；
    4. 品牌知名度不够，市场影响力不强；
  2. 欧梵的物流需求
    1. 提供24小时的全天候准时服务；
      1. 公司业务人员、机场和办事处相送人员与客户通信保持24小时畅通；
      2. 保证运输车辆24小时运转；
      3. 保证24小时内提货与交货；
    2. 服务速度快；
      对提货、操作、航班、派送都有明确的规定，时间以小时计算；
    3. 服务的安全系数高，对运输的全过程负责
      保证货物的各个环节都不出问题，一旦某个环节出了问题，将由本物流公司承担责任，赔偿损失；
    4. 信息反馈快
      做到对货物随时跟踪、查询，掌握货物运输全过程；
    5. 服务项目多
      根据货物流转的需要，通过发挥本物流公司的网络综合服务优势，提供包括出口运输、进口运输、国内空运、国内陆运、国际快递、国际海运和国内提供的派送等全方位的物流服务；
  3. 解决方案
    1. 仓储设置
      欧梵总体上形成了以广州市为中心仓库，各异地仓库为配送仓库的总体格局；
      1. 中心仓库设置：由欧梵派人管理库存；
      2. 各异地仓库设置
        根据欧梵的销售量，在全国划分为六大异地仓库即六个区域配送中心，各配送中心负责本大区的产品配送，各异地仓库通过以下两种方式来设立；
        1. 整合、收编该大区内一级代理商的仓库，由本物流公司进行管理和配送；
        2. 直接由本物流公司在设库当地选择合适的仓库作为配送仓库，并由本物流公司进行库存管理和配送管理；
    2. 运输供应商管理
      欧梵将所有干线运输和异地区域配送的业务统一外包给本物流公司，由本物流公司进行物流资源的整合；
    3. 加强物流信息化，建立以条形码为核心的信息系统；
      为配合欧梵的发展，本物流公司根据欧梵对条码的要求，开发物流管理系统。该系统包括调度管理系统和仓储管理系统两大部分，适用于广州市总部物流中心和各异地配送中心；
    4. 制定科学规范的操作流程；
      为满足欧梵的服务要求，设计并不断完善业务操作规范，并纳入程序化管理。都所有业务操作都按照服务标准设定工作和管理程序，制定出口、进口、国内空运、陆运、仓储、运输、信息查询、反馈等工作程序，每位员工、每个工作环节都按照设定的工作程序进行；
    5. 提供24小时的全天候服务；
      针对欧梵24小时的服务的要求，实行全年365天的全天候工作制度。周六、日（包括节假日）
    6. 提供门到门的延伸服务均视为正常工作日，厂家随时出货，随时有专人、专人提供和操作。在通信方面，从总经理到业务人员保证24小时的通信畅通，保证对各种突发性事件能迅速处理；
    7. 提供创新报务
      从欧梵的角度出发，推出新的、更周到的服务项目，最大限度地减少货物损失，维护欧梵的信誉；
    8. 充分发挥本物流公司的网络优势，提供欧梵的物流服务水平
    9. 实行全程负责制
      本物流公司出现的问题，积极主动协助欧梵解决，增加欧梵的物流服务水平；
  4. 所实现的效果
    由于欧梵在各地建立了商流和物流一体化的异地配送中心，将实现以下效果；
    1. 欧梵的服务可以直接影响到终端客户，可以摆脱代理商的制约；
    2. 企业的物流效率更高；
    3. 为解决欧梵销售通道狭窄的问题了实现条件，有利于欧梵的销售渠道从广度上进行拓展，从而达到欧梵的赵网络覆盖到终端客户的目的；
    4. 通过稳定部分进行异地配送，在车辆上和各异地配送中心做广告等方式，使欧梵的品牌深入人心，从而在全国范围内提高欧梵的品牌知名度；
    5. 将异地配送仓库设在配套厂周边，大大提高了欧梵对配套厂配套能力，缩短了欧梵的服务响应时间；
    6. 由于建立了高效、以条形码为核心的物流系统，各民地配送中心和总部物流系统形成了点到点的对接；
    7. 由于实现了销售物流统一外包给物流公司，一方面降低了欧梵的管理难度，另一方面有利于实现物流的规模化效应，为降低物流成本提供了有利条件；
