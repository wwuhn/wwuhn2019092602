物流信息

现代物流的重要特征就是物流的信息化，可以看作实物流与信息流的结合；在现代物流运作过程中，通过使用计算机技术、通信技术、网络技术等，使物流活动的效率和快速反应能力得到了提高；如果信息系统不畅或错误，会造成诸多的问题；为了使物流系统的各个环节各个部门紧密衔接，密切合作，必须通过信息的沟通，组织物流活动必须以信息活动为基础，是决策和调度的依据；

1 物流管理信息各子系统

1.1 订单处理子系统

1.1.1 订单受理（登记）

1.1.2 出库处理；如制作货物拣选明细，产生缺货信息报表，下达配送指示，产生装箱单、送货单等；

1.1.3 送货结束后的处理业务；如发送费用结算单据等

1.2 库存管理子系统：以最小的库存数量满足需求；库存管理包含两方面的含义：一是正确把握库存数量的“库存管理”；二是按照正确的数量补充库存的“库存控制”，也称为补充订货；

1.3 仓储管理子系统；

1.3.1 保管场所系统：进行货位的牌号标示，并在计算机信息系统内有体现出来，即使业务不熟悉的人员，也可以迅速、准确地进行货物拣选；

a 固定场所系统：便于作业人员的识别查找，但是货物的使用效率相对较低，当货物保管量小时，货位容易出现闲置，当货物量超出货位容量时，要采取其他措施弥补；

b 自由场所系统：

1.3.2 订货拣选系统；

1.4 配送管理子系统；根据运输合同生成运输任务，对每一个承运人生成运输任务交接单，同时打印送货清单、运单和运输标记；其次是运输过程管理，对于未完成的运输任务进行状态更新，生成报表，

1.5 运输管理子系统；

1.6 货物追踪子系统；

2 物流信息技术

2.1 电子数据交换系统（EDI）

2.2 条码技术 Bar code；设备用红处线或可见光源识别条码（条：光线反射率较低的部分，黑色的条吸收光；空：光线反射率较高的部分，空将光反射回扫描器），扫描器将光波转译成模仿条码中的条与空的电子脉冲，解码器将电子脉冲译成一种二进制码；

2.3 有线电子标签：装置于货架上的信号转换器、完成器、电子标签、订单显示器、现场操作计算机和服务器等一系列设备构成的网络化计算机辅助拣选系统；

2.4 射频技术：Radio Frequency Identification:非接触自动识别；

2.5 POS系统：销售时点信息管理系统

2.6 GPS技术：global Positioning System;

2.7 GIS技术：Geographical Information Sysstme 在物流中的应用有：车辆路线模拟、网络物流模型、分配集合模型（服务范围确定）、设施定位模型；
