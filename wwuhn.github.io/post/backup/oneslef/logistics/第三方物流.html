1 概念

第三方物流又称契约、外包、代理物流；Out-sourcing业务外包；是相对第一方发货方和第二方收货方而言的，是提供物流交易双方的部分或全部物流功能的外部服务提供者；第三方物流的服务范围不仅仅限于运输、仓储业务，它更加注重客户物流体系的整体运作与效率，它的业务深深地触及到客户销售计划，库存管理、订货计划，生产计划等整个生产经营过程中；

2 第三方物流兴起的原因

过去，大多数企业都是自己经营生产、销售业务，又拥有自己的车队、仓库，但是，物流属于费钱和劳力密集的活动，运送与储存物料以及产品分销占用大量空间、设备、人员，并愈加要求计算机软硬件支持，在当今严重的激励竞争环境下，提高资源的利用率成为企业生存的前提。随着现代企业生产经营方式的变革和市场外部条件的变化，工业型社会向信息型社会的过渡，企业从“纵向一体化”直接控制资源转向“横向一体化”，这就需要企业借助企业的资源来达到快速响应市场需求的目的，为此，企业将主要精力放在关键业务上，而将其非核心业务交给其它的专业企业于是，随着市场竞争激化和社会分工的细化，商家们开始思考空间是自营物流业务还是将物流业务外包出去。有些商家开始意识到自己并不是运输经营和库存管理的专家，为了把更多的精力集中到自己的主营业务上，以便同自己的对手竞争，有些厂商开始把一些自己不十分在行的，诸如运输、仓储这样的业务外包给第三方经营。

有需求就有供给，这是市场经济的基本规律，在电子商务环境里，消费者在网上虚拟商店购物，并在网上支付，这样，物流特别是第三方物注愈加重要；

3 第三方物流的效益原理

3.１　规模效益；

3.２　系统协调：是指第三方物流公司在自己所占有的供应商群及其各自的客户群之间进行协调，实现统一组织运输和配送；实现共同采购、储存和资源共享；

3.３　专业化：包括作业、设备、人的专业化；

5 第三方物流中影响客户服务的因素

5.1 服务可得性：运输网络覆盖程度，随时提供服务的能力

5.2 服务可靠性：装载前准确地通知发货人相关信息，准时发货，准时到达目的地，对在途货物全程跟踪能力，货物到达前准确地通知收货人到达时间，各项记录的准确性，货损货差率，服务质量的一致性、稳定性；

5.3 服务快速响应性：对客户的订货、查询快速响应，运输计划改变或运输延迟时快速通知发收货人，排除各种故障的快速性，对客户投诉做出快速反应，交货后快速将单据返回给发货人；

5.4 工作人员服务的专业性：工作人员的专业知识水平，处理业务问题的能力；

5.5 服务的完整性： 提供门到门服务，个性化服务，仓储、包装服务，对客户进行库存控制，为客户提供物流咨询、培训，交货后为收货人安装调试设备，帮助客户结算货款；

5.6 服务的一致性；

6 定制化服务模型

第三方物流企业根据自身的物流资源优势，对客户物流需求的细分，找出目标客户群，并通过对物流功能模块化和标准化，根据目标客户群的需求特征，提供一定规模的定制化客户物流服务，更好地体现物流服务个性化的这一特征；

7 快递

运营一个品质优良的快递公司不但必须具备其他优秀的第三方物流公司所应具备的条件，而且在程度上都要更进一步，特别是速度；

7.1 对于一个志向远大的国际快递公司来说，下面几个因素是区别于普通第三方物流公司的：

a 与国内大的传统运输企业联合，争取其庞大的网络网点优势；

b 在重点国际机场建立快递转运基地，保证货物中转快捷；

c 推出脍炙人口的服务项目，并有网络信息支持；

d 与海关密切合作，保证国际货物的迅速通关；

DHL(敦豪)：是一家建立27年的合资公司，网点多，有42家分公司，业务涉及318个城市，有自己的专用货运机；在际快递市场上能与DHL比肩的只有美国的UPS（联合包裹），FedEX（联邦快递），TNT（德国邮政）
