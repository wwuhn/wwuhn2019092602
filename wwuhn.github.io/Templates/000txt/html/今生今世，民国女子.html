今生今世，民国女子

张爱玲，民国时期的上海滩，她用文字书写了一段传奇。

如果把时光的镜头倒回到九十七年前的上海，一个小孩清脆的啼哭声划破中秋前的夜空，那时，深邃旷远的天空应该有一轮皎洁的月亮，静静地播洒着银色的光辉。

谁知道呢，兴许她来到这个仓皇的人世间，就是来还债的，就像《红楼梦》里的绛珠仙子黛玉是来还前世欠下的雨露恩泽，一生注定泪水涟涟。前世，她的欠下的债，今生，用至深的之情，笔耕之艰辛一一归还，待到还清的一天，云淡风轻，天地辽阔，她终于可以波澜不惊，静度余生。

张爱玲是一个天才，八岁的时候就动笔写小说，她的到来仿佛就是为了成就一场华丽的文学盛宴。人生这条路，风雨飘摇，遍布荆棘，她还很小的时候就已经懂得世情冷漠，人情淡薄，连至亲的亲人都不那么靠得住，她只有依靠自己。在莽莽苍苍的岁月中，她独自一人，尝尽辛酸苦楚，饱经风霜侵凌，清净干脆地描画着着她的文学梦。

张爱玲的作品里，时常会出现不一样的月亮，或晕染着，或清冷着，或遥望着，月亮不语，千百年来何曾有过改变，任凭沧海桑田，物换星移的变迁，她总是冷冷地高挂在墨色的天空，静静地散落清辉。张爱玲最广为流传的一张照片，她的神情就是这样的冷清倨傲，微仰着头，高高在上，俯瞰着这个烟尘滚滚的俗世，仿佛她素手一指，就能点破这离离红尘中的悲喜怨憎，爱恨离愁。

她从来清透，没有浮名之累，亦无可喜。对她来说清高到云端，卑微到尘埃都可以逆来顺受，无可指责。

张爱玲出生于一个没落的贵族家庭，她的曾祖父是昔日大清朝权倾朝野的大臣李鸿章。到她父亲这一代，已是夕阳无限好，只是近黄昏的落寞了。春日迟迟的童年，她接受了很好的古典文学的熏陶。老宅，深院，烟雾缭绕，她恍恍惚惚地站在满清遗老的面前吟诵“商女不知亡国恨，隔江犹唱后庭花。”

原本一家人可以靠着祖辈遗留下来的产业，衣食无忧，过着富足的生活。可是，他的父亲沉迷鸦片，浪迹烟柳巷，她的母亲又是一个新派女性，两人自是矛盾重重。终有一天，她这位很自我的母亲毅然出国，留下一对年幼的儿女不管不顾，到另一个崭新的天空自由翱翔。

接下来，那位父亲更是变本加厉，后妈，鸦片，搬家成了她儿时生活的主旋律，哪里还有丝毫温情。

所以，张爱玲从不同情，更不悲悯，别人觉得痛哭流涕的事情，在她的眼里也是清淡的一抹烟云，胡兰成说她“冷情”，其实她是看破了尘世的冷漠，即便她投之以木瓜，谁有真会报之以琼瑶？身世飘零，涕泪纵横，是因为还没看清，若真是看清其中因果，她的“冷情”又何尝不是另一种意义上的温情？

所有的苦难，属于你的，谁也背负不了，又何苦招摇着怜悯和同情于事无补？看似最冷漠的对待，其实是一种教育，让你看明白这个千疮百孔的世界不需要眼泪，也唯有坚强才能够独立风霜。

张爱玲，从来很坚强，很独立，她依靠文字而生，从不依附任何人。就是在她和胡兰成山盟海誓的那段美好的时光，她也没有让胡兰成小小破费。反倒是这个薄情的男人穷困潦倒的时候，她总不忘拿出稿酬，以备他不时之需。

遇到胡兰成是她命里的劫，她必须偿还的债。为他绽放为他枯萎，为他流泪为他伤痕累累，更像是她的命数。

初相识，胡兰成三十八岁，对于一个尝过世间百味的男人来说该是最好的年纪，正因为走过岁月荒芜，让张爱玲疏淡寂寥的心与他离的很近。在一个闲散的午后，胡兰成读到张爱玲的文字，他被她浓墨重彩的一笔撩动了情思，他的心瞬间被洗劫一空，从此，张爱玲成了他的意念里最真诚的向往。

当然，他这种向往并不只对张爱玲有过，在他或风光无限，或颠沛流离的一生里，从来不乏对各种风情的女人产生类似的向往。这些，张爱玲知道吗，也许多少有所耳闻。但是，当这个风流倜傥，谦谦君子模样的人站在她的面前，她还是心动了。他的谈吐，他的才华在张爱玲所有认识的男人里面定是最特别的，他们才有那么多话要说，一谈就是五个多小时。素未相识，却一见如故，每一段爱情都是从一句似曾相识的问好：“你在啊！”开始的吧。

文学的天地多么辽阔，多么温馨，成全了他们“桐花万里路，连朝语不息”的感情。然而，胡兰成爱她的文字她的思想胜过爱她的容貌。也许男人更倾向于视觉吧，可是，胡兰成是一个把玩文字的男人，还未见面就已经在她灵动的文字世界里缴械投降了。

在《今生今世》里，胡兰成有一段这样的描写：“我一见张爱玲的人，只觉得与我想得全不对。她进来客厅里，似乎她的人太大，坐在那里，又幼稚可怜相，待说她是个女学生，又连女学生的成熟亦没有。我甚至怕她生活贫寒，心里想战时文化人原来苦，但她又不能使我当她是个作家。”这是一种怎样的感觉，说不出是褒是贬，是震动是惊喜，抑或是她别样的气质让有种他闻所未闻，见所未见的的无措？就是这样，在你来我往的促膝而谈中，他们的确立了在彼此的心中的位置。

那个时候的胡兰成，事业正处于低谷，在文学的殿堂得以小憩，但这里不是他的归宿，这个自负狂狷的男人，是闲不住的，他有野心，执意要在外界风云变幻的世界里寻找隶属于他的一己之腾达。

用他的话来说，他贫寒的出身，此间经历的人情冷暖，早已经让他回复到天地不仁的境地，成王败寇，都可以淡然处之，也没有什么国家民族的是非观念。只要是利己的都是值得去做的。两厢厮守不久，他就出去奔前程了，偶尔回来和张爱玲小聚，其间他挟妓出游的欢场演出也从未间断过。

对于这些，她都置若罔闻，低到尘埃的爱情让原本面目狰狞的人变得可亲亦可喜。这段感情，一开始就是一个彻头彻尾的悲剧。

因为懂得，所以慈悲。胡兰成是懂她的吧，他说张爱玲是民国世界第一临水照花人，不用与世俗打交道，世界自会来与她交涉。如不是深深懂得，怎可能说出这样话，如此入木三分。

张爱玲也是爱他，宽容他的，她将自己锁在文字的小世界里不闻不问外漫天飞舞的流言蜚语，至少，在现实短暂的相会中可以和她深爱的人拥有凄凉的小团圆。可是，无望的爱情无尽的付出也是有期限的，她的退避，隐忍终于有一天会全盘颠覆，她不是任人摆布的棋子，她从来很骄傲。

张爱玲用了几年的时间来下定决心，最后的诀别，场景更是凄凉，他在逃亡的路上，暂避一隅，张爱玲不辞辛苦，从上海跑来看他，那时，胡兰成虽隐姓埋名在避难，他和一个姓范的女人早已同居。张爱玲来了，这个和他有婚书媒证的发妻，这个曾经他信誓旦旦要给她安稳现世，静好岁月的女人，他向人介绍她为妹妹。张爱玲怎会看不出他们常住在一起眉眼神情的相似，这样的屈辱忍受一次就足够了，也无多话，烟雨迷蒙中，她决然走了。

胡兰成在雨中送她，滔滔江水，无尽的哀伤，她的眼泪在他看不见的地方肆意地流淌，今生今世，他们缘尽于此了。这些决绝的话，在他落魄的时候是说不出的，此行一年多之后，张爱玲得知他基本安定，通过书信向他表明了态度。从此，转身天涯，永不相见。她的心自此是萎谢了，再也不能去爱别人，她把今生今世的爱都给了他，是错了，是醒了，终究是天涯陌路，毫无挂碍了。

后来她辗转去了美国，在那里邂逅了生命中的第二个携手风雨的男人。只是，这个叫赖雅的美国人也给不了她现世安稳，静好生活。更多是流离失所，居无是处，然而，张爱玲用她的顽强和坚韧毅力陪着这位暮年的老人走过了人生最温暖的十一年。

张爱玲是懂得的，这个世界的悲伤和苦楚以及茫茫中的无定数，渺小如尘埃的人啊，谁又可以说自己做得了主呢？所以，最执着的深情就是不惧风雨，不离不弃。她不是真无情，她只是用无情的面具来抵挡无尽的凄风楚雨，当不得不面对的时候，深陷其中的她不至于陷入绝望，不可自拔。

到了晚年，你决意隐居，不再和尘世中的任何人任何事有交集，她一生孤独，冷清，到了迟暮之年，她也只愿意守着这孤寂和她玲珑华彩的文字相伴余生。她是为文字而生的，她命中注定的遭遇是一种偿还，也是一种历练，当一切功德圆满，她也选择遗忘，遗忘前尘往事，一个人，异国他乡，等待命定的结局。她是如此聪颖，又是如此坚韧，虽然她喜欢市井的喧嚣，人间的气象，但是，当一切尘埃落定之后，她宁愿远避浮名，一个人的烟火人生也是一种活法。

也是在一个月夜吧，张爱玲在遥远的美国一所简洁的公寓里，独自一人收拾好自己的行装，穿上她最爱的一身旗袍，弃尘而去。她终于等到了命定的结局，一切结束都了，前尘往事如烟散去，只留下那些婉转凄迷的故事，久久地，久久地在人世回荡。
