现代人的40个失智时刻

现代人的智商，总量守恒。这次用超额了，下次就难免遭遇“失智”。

也许是生活压力太大、也许是真实的记忆早衰、又或是被其他事物分散了注意力。所以尽管大多数现代人表面上看着精明有序，但做起傻事来，倒也照样毫不含糊。

今天，我们整理了一些编辑们曾经遭遇的“失智时刻”，希望你在阅读时忍住别笑。毕竟，常在河边走，哪有不湿鞋呢？这里面说不定就有你吧。

白目

现代人的社交压力，有一半来自“如何更得体地与人交谈”。只可惜，听完了所有教人说话的高情商知识付费课，还是难逃大脑短路时的“语出惊人”。

1. 敲了敲门，对里面的人说了句，“哪位？”

2. 指着叶子说， “这绿子好叶”。

3. 误打误撞走进男厕所，看到里面出来的男人，指着他鼻子大喊：“你出去！”

4. 为了夸同事有才华，随口说到，“像你这么有才的人，根本不用care长得难看”。

5. 在工作群发通知，本想用“Hi guys”开头，结果打成了“Hi gays”。

健忘

上一秒要去做的事，下一秒就被忘得一干二净。现代人和鱼一样，每天都在上演“记忆只有七秒”的戏码，也许是因为海洋是一切生命的起源？呵，别再骗自己了。

1. 冬天穿了秋裤，都整理好了出门，到了电梯发现外裤没穿，穿着秋裤就出来了。

2. 有一天点了外卖，小哥咣咣咣敲门，我去开门，小哥告诉我“哎呀你点的餐我忘带了”。

3. 在梦里想到了选题，立马抓起枕边的手机，打开备忘录写了下来。醒来的时候，发现这根本也是个梦，手机里什么都没有，还把选题给忘了。

4. 宿醉醒来，发现近视四百度的自己看眼前一片清晰，以为是在醉酒过程中获得了某种超能力，遂疯狂与亲朋好友分享此消息，堪称人生奇迹。后来对着镜子仔细看，才发现是昨晚出门戴了隐形眼镜，喝醉后忘了摘。

5. 为了遛狗，一大早在小区逛了半个小时，才发现狗没带出门。所以我这半个小时到底在干什么？

6. 本人男，裸手吃了麻辣小龙虾，然后去上了厕所。

分神儿

自以为可以胜任多线程工作，但一个不留神儿，就会变成多线程翻车。

1. 看电影把手伸进了别人的爆米花桶里。

2. 从公司收发室拿到快递后，边走边拆，并在路过垃圾桶时顺便把买的东西扔了，然后手里把玩着快递盒子走回到工位。

3. 开车去看《小偷家族》，被感动得稀里哗啦，哭着走出电影院，默默打车回家了。

4. 在学校上洗手间的时候，旁边坑位的人问我用的什么牌子的纸巾，说是很香可不可以给他看一下，我想也没想就递过去了。后来，第二节 课我就没有再去上，一直蹲在坑里。

5. 拿着纸杯子去接水，然后顺便去厕所，结果一走神，直接把杯子拿到了厕所，只能放在洗手台旁边，从厕所拿着纸杯子出来的一瞬间，觉得自己在去做尿检的路上。

重点失焦

出于对仪式感和重要性排序的某种迷思，现代人总爱把最核心的事情留在最后。只不过，沙子越倒越多，当终于后知后觉时才发现，世上没有后悔药可以吃，瓶子里早就没了装石头的余地。

1. 第一次尝试炖排骨，从网上查到说需要生姜。偏偏家里没有，于是专门出去一个小时买来生姜（和其他一堆有的没的）。准备开火时，突然发现家里也没有排骨。

2. 心里想着去洗衣机里拿衣服出来晾，结果走到洗衣机前打开盖子，又往里倒了一堆洗衣液。

3. 作为新媒体编辑，想看推送预览，结果点成了“发送”，然后还云淡风轻地点击了“确认”。

4. 平时有出门前关空调的习惯，那天为了省电，一晚上都没开空调。早晨上班前，习惯性地用遥控器把空调打开，出门了。

5. 除夕夜，给全体微信通讯录好友群发了朴实无华的“生日快乐！！！”

非注意视盲

黑夜给了我黑色的眼睛，然后我就真的在黑夜里什么也看不见了。听上去好像很合理的样子。

1. 在餐厅吃饭，拿起桌角的调料盒，想倒孜然。结果把一堆牙签倒在了食物上。

2. 刷地铁卡进站的时候，刷了很久都没成功，直到后面的人提醒我，“你拿的是身份证”。

3. 和同事在餐厅吃饭，把旁边同事面前的小碟当成了烟灰缸，丢了一堆啃剩下的鸡骨头进去。

4. 早晨七点半还没完全清醒，拿起手机看了几眼就又睡过去了。九点钟醒了再看手机，刚刚在淘宝下单了一堆猫抓板。可是我没有猫。

5. 家附近开了一家影城，兴奋地打车去看电影，结果发现是婚纱影城。

6. 18号0:02的火车，18号晚上慢悠悠吃饭，收拾行李，晚上十一点准时到了火车站，时间安排简直太合理了。然后眼睁睁看着时间变成了19号。

7. 点完菜吃完之后，一直批评牛蛙肉又老又柴，结账时候才发现原来自己点了一盘小炒鸡。

常识盲区

没常识的生活太精彩了，每天都有惊喜，但更多的还是惊吓。

1. 发朋友圈：枝裕和的电影真的很不错。

2. 看到室友买了百合花，不知该说什么，于是尬聊到“等我们摆在客厅看两天之后，还能炒着吃”。然后我才在舍友咯咯咯的笑声中，知道了炒百合用的并不是百合花。

3. 几乎从没坐过地铁，那天心血来潮，走进地铁站还感叹环境舒适，连座位都像高铁一样。结果发现上的是机场专线，一口气25块坐到首都机场，最后又花60块打车回家。

4. 第一次领到工资后开始“理财”，在周五把所有钱都买了活期基金。结果发现是“T+1”日才到账，且金融市场周末休息。于是在发工资的第一个周末，度过了身无分文的两天。

5. 情怀满满地跑去看《XX公寓》，却坐在电影院看完一部盗墓片——如果买票前能刷刷朋友圈就好了。

都是手机惹的祸

有时候也会想，如果没有手机，生活该少多少麻烦？

1. 手机立在桌子上看电影，一边看电影一边还在拼命找手机，还怒斥女朋友是不是偷偷藏了我的手机。

2. 晚上在路边看到乞丐，想偷拍发微博表示怜悯，结果打开了闪光灯。

3. 想要群发，却给好友们拉了个群。

4. 为了逃避周末加班，回老板微信说“不好意思老板，我现在没有信号”。

5. 地铁马上到站，玩着手机站在门边等待。当意识到其实是对侧开门的时候，地铁已经再次发动了。

6. 第一次和客户见面，想要加微信，打开自己的微信付款二维码递给了对方。

都说成年失智最为致命，似乎这些智商停摆的瞬间，正是早衰显露出的蛛丝马迹。其实你大可不必担心，当失智时刻来临，拍拍自己的小脑门儿，一笑而过吧。毕竟，生活如此刻板紧张，不偶尔失智一下，又哪来的快乐源泉呢？
