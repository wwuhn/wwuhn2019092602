给大家盘点一下那些年我们看过的精彩犯罪电影

catch me if you can

小李子的杰作，又名《猫鼠游戏》。

根据根据小弗兰克·阿巴格诺的自传《有本事来抓我吧——一个诈骗犯令人惊异的真实故事》改编。

一个看似涉世未深的男孩，编造理由套现支票。

假装是学生记者套取资料，伪装身份获得制服后假扮成机长蹭飞以及伪兑支票。

用一串假冒的项链和虚伪的借口拈花惹草，甚至欺骗自己的父亲。

有些人，天生就是谎言之神的使徒。

2. 《愚行录》

Gukōroku

有时候寻找真相的人，往往是想掩埋真相的人。

男主角作为一个报社记者着手调查一个一年悬而未决的灭门案。

可是在调查的过程中各种隐藏的线索被慢慢的抽丝剥茧。

在这部电影里你能看到罗生门似的谎言；看到甚于白夜行的不伦之恋；

看到阶级的丑陋、自私；看到人性的怨念；看到愤怒的凶狠和残暴；

看到了原生家庭于人生的影响；也看到可怜之人的诉说欲望。

导演给了开放式结局，到最终也没交代真相。

不得不承认岛国拍起人性隐藏最深处那种阴暗残酷真是让人不寒而栗。

3.《燃烧》

原作是村上春树的《烧仓房》，同时结合了结美国作家威廉·福克纳的短篇小说《烧马棚》。

神秘男子Ben告诉男主自己有个烧塑料大棚的癖好，但其实他真正的癖好是定期杀害女人。

村上春树笔下的复杂时代特性转化成李沧东导演镜头里韩国青年被社会摧毁、压垮而扭曲的吊诡人性。

用愤怒畸形的火焰燃烧掉自卑，把犯罪情欲贫穷与富有，这些最后全都化为现世的祭品。

是被爱还是被孤独爱，是肉体上的陪伴，还是精神幻想中的完美意淫对象、伴侣？

4.《看不见的客人》

Contratiempo

经典佳作，全程高能，不断挖坑，不断反转。

该片构建了一出绝妙故事，将谎言这把双刃剑用到了极致。

乍一看整个进程是案情的不断重复推演，其实重点在于女律师与企业家之间的争锋相对，彼此间的智斗。

铺垫的细节和情绪的渲染恰到好处，案中案里隐藏的反转也排列得当。

一边是飞来横祸，丧子之痛，一边是层层谎言，欲盖弥彰。

整个故事毫不拖沓，悬念一直持续到最后一分钟。

更精彩的是如果你看过片子后再看电影海报，会发现海报上人物关系已经把结局剧透了。

5.《杀人回忆》

??? ??

韩国悬疑电影的经典之作，改编于真实案例。

当所有的证据都指向一个人时，突然来了一张DNA鉴定，却推翻了所有的证据。

主角不愿相信科学，想要杀了疑犯。

可若干年后，当男主角之一回到麦田，却发现其实凶手才刚来过。

电影结尾很有意思，万物复苏的季节，一个小女孩背着书包从稻田经过，根本不会想到这里就是轰动一时的命案地点。

男主问小女孩那个人长什么样呢？小姑娘眨着眼说道：很平凡。

他平凡得没有任何特征，那张极其普通的脸或许就是昨天擦肩的路人。

但我们永远不会知道。

6.《青蛙少年失踪事件》

根据韩国三大悬案之一的大邱少年失踪案改编而成。

和上面的《杀人回忆》一样，是一部看完之后让人既悲愤又无解的电影。

1991年五个小男孩上山玩耍后离奇失踪直到2002年才被发现尸骨。

在韩国因为此案的破案时间已经超出了死刑犯申诉时间的15年年限，所以成为了没有下文的悬案。

让人不禁唏嘘现实永远比电影更残酷。

7.《解除好友2：暗网》

互联网改变世界，也在悄无声息改变你的生活。

电影采用的桌面拍摄方式很新颖，男主角因为好奇心解锁了一台捡到的苹果电脑。

当他通过这台电脑和好友们联机视频后，他们逐渐发现了潜藏在这台电脑背后令人毛骨悚然的秘密。

暗网呈现了一个庞大的黑客世界，当你凝视深渊的时候，深渊也在凝视你。

整部片子没有任何的恐怖元素，但是却能让你不寒而栗。

8.《心迷宫》

称得上国产悬疑犯罪的佳片。

在国内的一个不知名偏远小乡村，因为一具无法查明身份的尸体而引出了一系列离奇古怪的故事。

罗生门地将这个小村子变成了一个压抑不安让人心生罪恶的封闭空间。

电影反映了农村闭塞环境下人性的险恶.

9.《龙纹身的女孩》

大卫·芬奇执导的一部惊悚悬疑电影，改编自瑞典推理小说名家斯蒂格·拉赫松《千禧年》三部曲的第一部。

讲述一位神奇的女黑客和一位男记者合作破案的故事。

扑朔迷离的案件，冰冷诡异的气氛。

导演鬼才般的拍摄功底在这部作品中将犯罪悬疑玩转得游刃有余。

大量的信息抽丝剥茧随着剧情的推进而一一解开，毫无拖泥带水的节奏把控更给人以酣畅淋漓的惊艳。

连环杀手、虐待、反社会人格、性变态......

犯罪电影多数血腥猎奇引人入胜，可是要知道现实可比电影来得残忍。

我们用观众的视角观看犯罪电影固然有趣。

可是人性的底线，善恶的维度，或许才是犯罪电影真正应该表达的意义所在。
