修理时，要先把价格谈好，并在施工单上写清楚，如另有毛病需要修理，则需要求有通知。并要求看到换下来的零配件。

1、换机油。

一辆车的机油用量大概会在4到5升之间，一般都是处于一桶不够，两桶富裕的尴尬境地。这就给维修工创造了条件，用不完的机油一般会想办法留下不让消费者带走，这就便于下个客户来之后使用之前的剩油。

2、机械方面。

机械是最「肥」的地方，因为机械器件的好坏，是否翻新，不懂行的你根本就看不出！在有些产品上，取了好的部件，结合在一起组合成新的，再清洗、去伤，在有车要换这个件的时候给用上！

3、电路。

电路是说那里有毛病，那里就有毛病。缸线100多一条，看着差不多了就提出更换。有钱人不在乎啊，一口气就4条，好的留下来加工给别的车。4、刹车。

当发现车子的制动力下降后直接建议你更换刹车片，或者刹车碟。其实只要清洗剂洗一洗，换换制动液就能搞定。

5、轮胎。

对于那些需要在修理厂过夜或者不懂车的小白同志要注意了，在你不经意间他们会给你的轮胎做些手脚，低劣的是给你胎壁内侧划扣子，或者干脆给你换一条同型号的事故胎。

6、钣金喷漆。

所谓的原厂漆，高温烤漆是不存在的，能达到原厂的效果修理店是不会有那种设备的。只不过是给你个好听的说辞而已，让车主多掏钱。

7、保险杠等塑料配件。

这类产品，撞凹了就告诉你要更换，其实原厂保险杠都是PP材质，这类材质其实可以热修复的，小的碰撞只需修复就可以（比钣金还容易）!

8、洗车。

反正再好的车漆他们都能给你找到各种吓人的缺点，无非就是推荐各种保护用品。其实打蜡镀晶这种东西还是少做为好，毕竟原厂车漆真的没你想像的那么脆弱!

以次充好

有装车件，正厂件，厂家指定配套件，市场配套件，“三无”的非配套件。

故意制造故障

如故意少加机油，一段时间后造成发动机烧缸。

偷工减料

如只换了外面的3个火嘴，而收6个火嘴的费用；

只换不修

其实有些零配件是可以修理的；

小病大医

如ABS故障灯，可能只是轮胎部位的传感器需要清理，而并不需要更换电路板；

如离合器分离清，可能只是离合器泵里的垫圈的问题，而不需要换离合器；
