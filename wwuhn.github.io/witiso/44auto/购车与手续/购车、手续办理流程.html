购车、手续办理流程

  1. 买车注意事项
    1. 购车预算：级别选择，量力而行；
    2. 二手车辆，可以考虑；
    3. 汽车排量，够用就行：在满足动力性和加速性要求的前提下，尽量选择排量小的车型，因为一般情况下排量越大、油耗越大；家轿的黄金排量在1.6-2.0L;
    4. 次要配置，可有可无：主要要考虑发动机、变速箱、底盘三大件；凡是涉及车辆舒适、豪华、便利配置，如电子地图、车载电话、真皮座椅、安全气囊、CD音响、铝合金轮毂、金属漆、防撞侧杆、电动后视镜、电动门窗、顶窗、防盗设施等可以在用车过程中逐步增加，不一定非要买车时一步到位；
    5. 即将停产或未热销的车型，尽量不要购买；
    6. 常识须知和参数解读：
      1. 发动机排量：排量越大，动力越好，但油耗也越大；
      2. 发动机最大功率[kW(ps)/rpm]：如123(167)/6000，当一辆车的最大功率表现在较纸转速时，稍提转速，功率会迅速增大，说明该车回还性能好，很容易超车；
      3. 发动机最大扭矩(N·m/rpm)：如245/1400-4000，当一辆车的最大扭矩出现在较高转速时，则表明这辆车的后备功率大，在高速行驶时负荷率低，燃油经济性要差些；
      4. 日常转速范围大多为2000－4000r/min，如果最大功率和最大扭矩在5000r/min才可以达到，则很少有机会可以用到其最大功率和最大转速；
      5. 低配与高配的区别：在同一动力的前提下，低配与高配的发动机、变速箱、底盘等关键的核心部件都是一样的，所以区别主要是车内配置的差异，与品质无关；如音响、天窗、导航、座椅面料、自动空调等；有些车型，在安全防护上也会有些差异，比如增加了车身稳定控制，多了几个气囊等。
  2. 购车时机的选择：7、8月的车展、月末、季末（4S店要冲返点的销售额）、非节假日；
  3. 根据自己的用途选择车种和类型；
  4. 总价预算与轿车级别选择；
  5. 车身结构的选择：三厢式、两厢式；
  6. 驱动方式的选择：前置前驱、后置后驱、前置后驱、四轮驱动；
  7. 变速箱操作方式的选择：手动档、自动档、手自一体；
    8. 项目 手动变速箱 自动变速箱
    优点 购买成本低、使用成本低、山区驾驶有优势、节省燃油、更易掌控汽车； 使用便捷、拥堵路段有优势、双离合变速器的效率高；
    缺点 需要相应的驾驶技术、频繁起步比较辛苦； 购买成本高、需要定期换油；
  8. 车型改款次数：改款换代次数高的车型具有一定优势；
  9. 保有量和销售量：二者都大的话则有较好的保值，以及维修的便利与实惠；
  10. 选择品牌、车系、车型：要尽量收集车型的口碑资料；
  11. 保值率与新车故障率的了解；
  12. 了解具体车型的厂商指导价和全国最低价；
  13. 经销商的选择：汽车交易市场、品牌专卖店、汽车经销商的4S店、综合经销商、网上商城；
  14. 选择是否分期：全款或银行或汽车金融公司贷款，按揭购车、三思而行;
  15. 谈价、砍价：
    1. 肢解价格：通过销售人员所给出的价格，除去购车税、上牌费、保险费等费用，算出的价格再与网上报价作比较，就能判断可砍价格空间。(4S店一般的作法是给一个价格，减个折扣，大概2.5w，然后再加上购车税、上牌费、保险费、分期手续费等费用，大概也是2.5w；)
    2. 货比三家：首先消费者应该避开离家最近的4S店，到市内其他4S店进行砍价，走访2-3家即可。之所以要避开离家近的4S店，一方 面是该店方便自己购车；另一方面是避免二次进店谈价、砍价所造成的尴尬。随后，按照各大网站价格前往第一家4S店进行砍价，再按照最后拿到的价格，接着往 下砍，基本上到了第3家，就能看出某款车型的可砍空间是多少。最后再到离家较近的4S店进行谈价、购车。
    3. 配件尽量少买；
    4. 不要过早声明现金付款、不要过于关注月供；
  16. 验收新车：

    查外观、看内饰、试开车、听声响、数配件、验文件；

  17. 办理保费、缴纳税费、验车选号上牌

    工商局（验证）→车税局（交购置税）→保险公司（购买保险）→机动车检测场（验车）→车管所（领牌照、行驶证和登记证书）→地税局（交车船税）→经销商（告知牌照号码，建立国主档案）

    1. 带购车发票、汽车的出厂合格证、本人身份证到当地工商局的机动车市场管理所盖工商验证章。（只盖个章）
    2. 联系保险公司或者是通过车辆销售点联系保险公司办理交强险，必要时可以上全险。
    3. 缴纳购置附加税：带齐车辆合格证+购车发票+身份证（如果是进口车辆，需提供海关货物进口证明书、商品检验单），每样复印2－3份。到国税局缴税大厅去缴纳车辆的购置附加税。购置附加税=购车款/(1+17%)×购置税率；购置税率 : (1.6及以下为5%，其它为10%)
    4. 去当地的税务局缴纳车船使用税，领取字牌。
    5. 办临时号牌：带身份证，购车发票，交强险（原件、复印件）去车管所或交通支队办临时号牌，临时号牌可办理3次。一次有效期7天。注意需在有效期内验车（各地情况稍有不同，请咨询当地车管部门）
    6. 车管所车辆检测：带车辆合格证、车辆发票四联、购置税完税证明、交强险保险单正本、身份证及复印件、车主驾驶证到当地车管所进行车辆的检测，检测合格后可以拿着回执去选号啦~
    7. 选号：在车管所还要进行车牌照的选号，
      网上选号：自选车牌都在车管所网站上进行。由于各地选号政策方式不同，请选号前仔细了解当地选号规则。
      现场选号：随机号牌可当场领取选好（电脑随机选号），号牌可邮寄，也可返回现场领取；
    8. 选号后再在车管所办理车辆行驶证，领取机动车登记证书，这样领到车牌之后装上并将保险贴验车贴都贴在车前挡风玻璃的右上角就可以自由驾驶了。
